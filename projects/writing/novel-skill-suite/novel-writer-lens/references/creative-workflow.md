# Creative Workflow

## When to use

- The user already has `writer-assets` and wants a new story plan or chapter draft.
- The user has only a premise and needs a clean planning pass.
- The user wants the old `chinese-novelist` flow without the duplication and rigidity.

## Distilled intake

Ask only for missing high-impact inputs:

1. Genre and tone
2. Protagonist setup
3. Core conflict and stakes
4. Length target or chapter count
5. Constraints, taboos, or reference style

If `writer-assets` already exist, skip questions that the assets already answer.

## Planning order

1. Build a one-paragraph project brief.
2. Select or generate role cards.
3. Build an outline with opening hook, escalation, reversals, and payoff.
4. Define scene functions for each chapter.
5. Define pacing and chapter-length targets.
6. Draft and revise against the quality gates.

## Drafting loop

- Write from beats, not from prose imitation.
- Give each chapter a clear event, a pressure source, and an end-state change.
- Keep chapter endings open unless the user wants a closed ending.
- Revisit the outline only when a chapter reveals a real structural problem.
