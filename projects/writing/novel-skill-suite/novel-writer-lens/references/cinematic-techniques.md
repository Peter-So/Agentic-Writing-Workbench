# Cinematic Writing Techniques

Use these when the user wants stronger picture sense, scene movement, or film-like pacing.

## Core techniques

### 1. Shot order

- Start with establishing space.
- Move to medium action.
- End on detail or reaction.

### 2. Motion grammar

- Use push-in for attention.
- Use pull-back for reveal or scale.
- Use follow-shot for pursuit or tension.
- Use cut-like paragraph breaks for turn or contrast.

### 3. Light and color

- Give each scene a dominant light source or color temperature.
- Use contrast to mark moral pressure, memory, or threat.

### 4. Sound and texture

- Let one or two sounds anchor the scene.
- Use material texture to make spaces feel touchable.

### 5. Montage logic

- Place two images side by side when the contrast matters.
- Use quick alternation for urgency.
- Use slow sequencing for dread, awe, or intimacy.

### 6. Close-up logic

- Use hands, eyes, weapons, letters, rings, wounds, or props to carry emotion.
- Make the object change reflect the emotional change.

### 7. Internal camera

- For inner scenes, keep the view close to the character's current perception.
- Use memory fragments, sensory triggers, and delayed realization.

## Practical mapping

- **Opening**: wide shot -> character entrance -> key object close-up.
- **Dialogue scene**: space -> placement -> gesture -> line -> reaction close-up.
- **Action scene**: threat -> movement -> contact -> consequence -> aftermath.
- **Inner scene**: trigger -> perception blur -> memory splice -> current choice.

## Do not overuse

- Do not turn every paragraph into a camera note.
- Do not sacrifice causality for visuals.
- Do not copy the template literally if the scene wants a different rhythm.
