# 角色认知真实性原则

角色在任何时刻说的话、做的判断，必须严格限定在**他此刻实际拥有的信息、经历、情绪状态**之内。这是最容易被忽视、也最容易破坏沉浸感的问题。

---

## 核心原则：角色只能知道他知道的

写每一句对话或内心活动之前，先问：

> **这个角色，在这个时间点，凭什么知道这件事？**

如果答案是"因为作者知道"，那就是问题所在。

---

## 四类常见违规

### 0. 预知模式（作者借角色发出叙事信号）

**问题**：角色"不知道为什么"就记住了某件事，或"眼睛先知道，脑子还不知道"——这是作者在用角色向读者暗示"这个细节将来有用"，不是角色的真实认知。

❌ 错误示例：
```
他不知道为什么要记住，就是记住了。
有些东西，眼睛先记下来了，脑子还不知道为什么。
你的眼睛先知道，先记下来了，等到哪天用得上，就用得上了。
他不知道为什么，总觉得那片湖底下压着什么东西。
```

问题：这些句式的共同结构是"角色有感知，但没有来源"——感知本身是作者给的，不是角色经历来的。读者感受到的是作者在操控，不是角色在活着。

✅ 正确做法：记住某件事，必须有具体的感知来源

```
# 错误：神秘预感
他不知道为什么，总觉得那片湖底下压着什么东西。

# 正确：经验来源
骚皮在这里陷进去过，他把这个地方记住了，记住了那股气味，记住了那片白。

# 错误：叙事信号
他不知道为什么要记住，就是记住了。有些东西，眼睛先记下来了，脑子还不知道为什么。

# 正确：具体细节触发
那个人站着的方式不一样，背挺得很直，手放在身侧，不动。草场上，不认识的东西，先记住。
```

**修法**：
- 删掉"不知道为什么"，改为写出触发记忆的具体细节（异常的姿态、气味、声音、行为）
- 删掉"眼睛先知道"类的元叙事解释，改为角色的实际经验逻辑（危险的地方要记住、不认识的东西先记住）
- 父亲/长辈的教导可以保留，但内容必须是实用的生存经验，不能是"你的眼睛先知道"这种神秘化表述

**重复检查**：同一个"不知道为什么记住了"模式，全书只能出现一次。多次出现等于作者在反复提醒读者，破坏沉浸感。

---

### 1. 突兀的目的地/方向

**问题**：角色在没有任何信息来源的情况下，明确说出目的地。

❌ 错误示例：
```
牛阳（12岁牧羊娃，从未离开边境）说："往东，去中原。"
季常锋（刚逃出将军府，包袱还没打开）说："往中原跑。"
```

✅ 正确做法：
- 目的地必须通过故事事件逐步建立：先逃、再被引导、再知道方向
- 没有信息来源时，只写本能方向（"往东""往黑暗里"），不写具体目的地
- 目的地由外部角色告知、或打开包袱/信件后才能知道

---

### 2. 超出认知的战略分析

**问题**：角色对局势的分析超出其年龄、经历、当前情绪状态所能支撑的范围。

❌ 错误示例：
```
麹明珠（刚逃出来的公主，惊魂未定）说：
"西域那边的人找我，不是因为我是公主，是因为我是高昌族剩下的血脉，
是高昌残留势力可能汇聚的一面旗，我活着就是一个目标。"
```

✅ 正确做法：
- 把完整的战略推断拆成**碎片**：她只知道父亲的警告、模糊的危险感
- 用"不知道为什么，但知道不能停"代替清晰的政治分析
- 完整的认知可以在后续章节随经历积累后逐步呈现

---

### 3. 超出当前情绪状态的冷静判断

**问题**：角色在极度恐惧/悲痛/混乱时，做出需要冷静头脑才能做出的判断。

❌ 错误示例：
```
麹明珠（刚逃命，在花船待了半年）：
"被看见有时候比躲着更能引出想看见的人。"
（这是反侦察策略，需要相当的心理成熟度和战略思维）
```

✅ 正确做法：
- 逃命状态下的人只有本能：跑、藏、不知道下一步
- 战略思维是后天习得的，需要在故事中有明确的成长节点
- 如果角色需要有这种判断力，先写出她是怎么学会的

---

## 检查方法：认知来源追溯

对每一句"超出预期"的对话或判断，追溯：

```
角色说了X
→ X需要知道Y
→ 角色在此之前有没有获得Y的信息？
  → 有：合理
  → 没有：需要修改
```

**修改方向**：
- 降低确定性（"我不知道，但感觉……"）
- 改为碎片化感知（"父亲说过……但说的什么，记不清了"）
- 改为本能反应（不分析，只行动）
- 把完整认知推迟到角色有足够经历之后

---

## 空洞动作的替换原则

"眨了眨眼""摇了摇头""点了点头"这类动作本身没有信息量，是AI写作的典型痕迹。

**替换公式**：内心活动 + 表情变化 + 身体动作

❌ 空洞：
```
牛阳眨了眨眼。
```

✅ 有层次：
```
牛阳没有立刻回答。他看了她一眼，那双眼睛里有什么东西在转，
不是不信，是在想这件事落到自己身上意味着什么。
```

**常见空洞动作及替换方向**：

| 空洞动作 | 替换方向 |
|---------|---------|
| 眨了眨眼 | 内心活动（在想什么）+ 眼神变化 |
| 摇了摇头 | 为什么摇头 + 身体其他部位的反应 |
| 点了点头 | 是真的同意还是敷衍 + 表情 |
| 挠了挠脖子 | 这个动作背后的心理状态 |

---

## 空洞情绪标签的替换原则

"眼睛里有那股劲""眼神憋着""眼神等着"——这类写法用抽象标签代替了具体感知，读者无法在脑子里形成画面。

**根本问题**：情绪标签（劲/憋/等）是作者的结论，不是读者能看见的东西。

**替换公式**：把情绪拆解成可见的身体部位 + 具体状态 + 行为倾向

❌ 空洞情绪标签：
```
眼睛里有那股劲，憋着，等着。
```
→ "劲"是什么？"憋"是什么样子？"等"是什么姿态？读者看不见。

✅ 拆解成可见细节：
```
他的身子往前倾了一点，两手按在膝盖上，指节压白了，
嘴唇收着，像是有话要说，又知道这时候不该说。
```
→ 身子前倾（急迫）、指节压白（克制的力道）、嘴唇收着（压住的话）——三个细节叠加，读者自己感受到那股憋着的劲。

**拆解方向**：

| 情绪标签 | 可见的身体细节 |
|---------|-------------|
| 眼神有劲/眼神锐利 | 身子前倾、手收紧、呼吸变浅、视线不移开 |
| 憋着/压着 | 指节压白、下颌收紧、嘴唇抿住、手按在某处不动 |
| 等着/期待 | 身子微倾、脚尖朝向目标、呼吸放慢、眼睛不眨 |
| 愤怒/压火 | 鼻翼动了一下、手握成拳又松开、说话变短、停顿变长 |
| 不安/紧张 | 腿抖、手摸某个习惯性的东西、眼神扫门口、话说快了 |
| 轻蔑/不屑 | 眼皮抬了一下、嘴角动了一下、没有转身、回答晚了半拍 |

**原则**：一个情绪，至少用两个不同部位的细节叠加表现，不能只靠眼神。眼神是最容易被滥用的单一细节。
| 耸了耸肩 | 是真的无所谓还是掩饰 |

## 观察者认知边界

旁观角色（如佛子观察常心）的推断，也必须有依据。

❌ 过度推断：
```
佛子一眼就看出：这是一个把某件事压进了骨头里、
暂时不打算挖出来但也没有忘记的人。
```

✅ 基于细节的推断：
```
佛子注意到：步子落地有分量（练过），右手进门时往腰间靠了一下（习惯摸刀）。
这些细节拼在一起，只能说明这个人不完全是来修行的，其他的，他不知道。
```

**原则**：观察者只能从**可见的外部细节**推断，不能直接读心。推断越精准，需要的依据越充分。

---

## 指代不明的心理描述

角色听到/看到某句话后的内心反应，必须明确指向具体内容，不能用模糊的"那个字""这两个字""某种感觉"代替。

**常见违规模式**：

❌ 错误示例：
```
"我来处理，"曹龚说，"你不用管。"
季常锋的心跳快了一拍。
那个字在脑子里转了一圈，转不出去，卡在那里。
他听见这两个字，反应太多，一下子堵在一起，动不了。
```

问题：
- "那个字"和"这两个字"指代不明——前文是整句话，没有单独的字
- "转了一圈转不出去"和"反应太多堵在一起"重复且模糊
- 没有写出角色真正的认知状态

✅ 正确做法：
```
季常锋的手指收紧了。
"将军那边"——曹龚说"我来处理"。这句话的意思，他听懂了，
只是那个结论太重，落进脑子里，一时砸不到底。
```

**修复原则**：
1. 明确写出角色听懂/看懂了什么（具体内容）
2. 用身体反应代替"心跳快了一拍"等空洞生理描述
3. 心理状态用"结论太重落不下去"而非"反应太多堵在一起"——前者是认知，后者是模糊感受
4. 不要用两句话重复同一个模糊状态

---

## 循序渐进原则

信息、目的地、计划，必须通过故事事件逐步建立，不能凭空出现。

**正确的信息获取路径示例**：

```
牛阳的逃亡路线：
第1章：只知道往东跑（本能）
第2章：到了边境黑市，听说中原
第3章：遇到老乞丐，被指引去洛阳
第4章：才真正知道目的地是洛阳
```

每一步都有信息来源，不跳步。

---

## 角色立场一致性：行动必须从自身利益出发

> 强势角色不会主动暴露对自己无威胁的目标的价值。角色的每一个判断和行动，必须从他自身的利益、立场、认知出发，不能因为"剧情需要"就让角色说出超出其立场的话。

### 常见违规：强势角色替剧情做信息传递

❌ 错误示例：
```
阿鲁台说："跑出去的，我们会处理。但有一个人，不能跑出去。"
中原人说："高昌族的公主。"
阿鲁台说："对，她不能活着出西域。"
```

问题：阿鲁台是西域枭雄，他要的是高昌族的地、马场、盐道。一个手无缚鸡之力的公主对他的实际统治构不成威胁——他没有理由把她列为"必须处理"的优先目标。这是作者借阿鲁台之口，替读者传递"公主很重要"的信息，不是阿鲁台自己的判断。

✅ 正确做法：让有理由在乎公主的那一方先提，让没有理由在乎的那一方质疑：
```
中原人说："高昌族的公主，也在你们的处理范围里？"
阿鲁台说："一个女孩，能翻出什么浪？"
中原人说："她是高昌族王的血脉。活着，就是一面旗。旗插在哪里，人就往哪里聚。"
```

逻辑还原：
- 中原人有政治顾虑（公主逃入中原会成为舆论把柄，或被人借名义生事）→ 他提
- 阿鲁台的逻辑是实际控制（地盘、资源）→ 他质疑
- 中原人解释政治逻辑 → 阿鲁台才被说服或保持沉默

### 检查方法：立场还原

写完任何一句重要对话后，问：

```
这句话是谁说的？
→ 这个角色，在他的立场和利益框架里，有没有理由说这句话？
  → 有（符合他的利益/认知/情绪）：合理
  → 没有（是作者借他的嘴传递信息）：需要修改
```

### 常见违规模式

| 违规类型 | 表现 | 修法 |
|---------|------|------|
| 强势方主动暴露弱势目标的价值 | 枭雄主动说"那个女孩必须死" | 让有政治顾虑的一方先提，让枭雄质疑，再由提出方解释 |
| 反派主动解释自己的弱点 | 反派说"我唯一的弱点是……" | 让主角通过观察/试探发现，不让反派自己说 |
| 角色替对手说出对手的优势 | 我方将领说"敌军的骑兵无懈可击" | 改为我方将领说"上次正面冲，死了三百人" |
| 无利益关联的角色主动关心某事 | 与某事无关的人突然说"这件事很重要" | 只让有利益关联的角色主动关心 |
| **高位者点名具体目标** | 将领/首领说"那个公主，我来处理" | 高位者按类别和行动思考，不按个人目标思考——改为"收尾的事我来"，具体目标由有个人动机的角色（仇恨/执念/命令）才能点名 |

角色面对问题时，不能直接"知道"答案在哪里。行动必须经历：**尝试→失败→推断→寻求帮助→再行动**的完整链条。

### 常见违规：上帝视角定位

❌ 错误示例：
```
牛阳发现骚皮不见了，往东边看了一眼——碱湖在那个方向。
他去把短剑取来，出了门。
```
→ 一个12岁的放羊娃，凭什么一眼就知道羊跑去了碱湖？这是作者知道，不是角色知道。

✅ 正确做法：让角色按照他的经验和认知，一步一步找：

```
行动链示例（找走失的羊）：
1. 先去羊的习惯地点找（草密的地方、水源边）→ 没找到
2. 看地面蹄印，但蹄印乱，分不清方向 → 推断出了事
3. 想到狼的威胁 → 心里有了紧迫感
4. 意识到自己没有牧犬，去借 → 寻求帮助
5. 牧犬追踪气味，引出方向 → 才知道往哪走
6. 跟着走，看见地面变白，认出是碱湖 → 才有背景知识的触发
```

### 检查方法：行动来源追溯

对每一个"角色知道去哪/做什么"的时刻，追溯：

```
角色做了X（去某地/采取某行动）
→ X需要知道Y（某地有羊/某方向有线索）
→ 角色在此之前有没有获得Y的信息？
  → 有（蹄印、气味、别人告知）：合理
  → 没有（作者直接给了答案）：需要修改
```

### 经验型角色的行动逻辑

有经验的角色（老牧人、老猎手、老兵）可以比普通人更快推断，但推断必须基于**可见的具体细节**，不能凭空"感觉到"。

✅ 有经验的放羊娃：
- 知道羊的习性（爱去草密的地方、水源边）
- 会看蹄印（大小、深浅、步距判断是哪只羊）
- 知道这一带有狼，知道小羊最危险
- 知道牧犬能追踪气味，知道去哪里借

❌ 不合理的"经验"：
- 没有任何线索，直接"感觉"羊在某个方向
- 一眼看出复杂的情况，没有推断过程

### 背景知识的触发时机

角色对某个地方/事物的背景知识，应该在**他接触到那个地方/事物时**才触发，不能提前出现。

❌ 错误：角色还没到碱湖，就先介绍碱湖的危险性
✅ 正确：角色走到碱湖边，看见那片白，才想起小时候阿爸说过的话
