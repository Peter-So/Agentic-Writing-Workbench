# 质量检查清单

交付章节前使用此清单确保质量。

## 整体检查

### 基础要素

- [ ] **章节有明确标题**
  - 标题与内容相关
  - 吸引人但不过度透露

- [ ] **字数符合预期**
  - 常规章节：4000-5000 字
  - 低于 3600 字必须新增关联情节补足动机、行动和细节
  - 高于 5500 字必须精简无关情节

- [ ] **章节完整性**
  - 有开头、发展、高潮
  - 不是片段，是完整叙事单元

- [ ] **时间地点清晰**
  - 读者知道何时何地
  - 转换时有明确标记

---

## 开头检查

- [ ] **前 3 段内抓住读者**
  - 有行动/冲突/悬念
  - 不是天气或日常流程

- [ ] **与上一章有连接**
  - 回应上一章结尾
  - 或明确时间/地点跳跃

- [ ] **背景信息不过量**
  - 没有大段信息倾倒
  - 信息自然融入动作

---

## 内容检查

### 情节推进

- [ ] **本章有核心事件**
  - 发生了不可删除的事
  - 不是"什么都没发生"的过渡章

- [ ] **推动主线剧情**
  - 揭示新信息
  - 或改变人物关系
  - 或升级冲突

- [ ] **逻辑自洽**
  - 事件因果关系合理
  - 没有巧合驱动剧情
  - 人物行为符合动机

### 冲突与张力

- [ ] **有明确冲突**
  - 人与人、人与环境、人与自己
  - 冲突推动本章事件

- [ ] **张力有变化**
  - 不是平铺直叙
  - 有紧张和缓解的交替

- [ ] **有转折或新信息**
  - 不是线性可预测
  - 有意外或新发现

---

## 人物检查

- [ ] **人物行为一致**
  - 符合已建立的性格
  - 如不一致，有解释

- [ ] **人物有反应**
  - 对事件有情绪/行动
  - 不是被动道具

- [ ] **人物有声音**
  - 对话能区分角色
  - 每人说话方式不同

- [ ] **人物展示而非讲述**
  - 通过行动/对话表现性格
  - 不是直接陈述"他很勇敢"

- [ ] **角色认知边界**
  - 角色说出的目的地/计划，在此之前有信息来源
  - 角色的分析判断，符合其年龄/经历/当前情绪状态
  - 逃命/极度恐惧状态下的角色，不会做出冷静的战略推断
  - 旁观者的推断，基于可见细节，不直接"读心"
  - **称呼与真名链路**：角色只能知道自己实际听见或已被告知的称呼。若场上只出现“曹大人”“那位贵人”“青衣客”“后厨呆子”，旁观者不能自动替换成“曹龚/真实姓名/门派身份”。必须先由知情角色追问、确认、回忆旧线索，或保留为含糊称呼。
  - 详见 [character-cognitive-authenticity.md](character-cognitive-authenticity.md)

---

## 对话检查

- [ ] **每句对话有目的**
  - 推动情节/揭示人物/制造冲突
  - 没有"你好""吃了吗"等无意义对话

- [ ] **对话简洁自然**
  - 删除冗余词语
  - 符合真实说话方式

- [ ] **有潜台词**
  - 不是所有话都直说
  - 有言外之意

- [ ] **问话符合角色位置与场景礼法**
  - 角色问出口前，先判断他有没有资格问、敢不敢问、为什么现在问
  - 普通人/弱势者/陌生人避免突然抛出审讯式短问（如“法号？”“身份？”），除非场景本身就是盘查、审讯或压迫
  - 试探、寒暄、遮掩、退让要符合人物身份；香客问僧人可用“这位师父怎么称呼”，官兵盘查才可用“叫什么”
  - 短句要有关系支撑：上位者、熟人、危急时刻、情绪失控时可以短；礼貌场合和陌生关系中需有动作、称谓或缓冲
  - 检查这句话是角色在说，还是作者为了推进信息逼他说；若删掉角色动作和处境后只剩“索要信息”，就要重写

- [ ] **标签使用正确**
  - 能辨识时省略标签
  - 不过度使用副词

---

## 悬念检查

- [ ] **结尾自然落地**
  - 本章情感或行动有完整的落点
  - 不因"必须设钩子"而硬塞突发事件

- [ ] **钩子（可选）**
  - 如使用悬念钩子，强度与章节位置匹配
  - 不是虚假悬念（误会型/无意义"突然"）
  - 下一章会回应这个悬念

---

## 情节波折检查

详见 [plot-tension.md](plot-tension.md)

- [ ] **本章主要行动有真实阻力**
  - 不是第一次尝试就完全成功
  - 阻力来自外部障碍、内部动摇或信息不足，而非假悬念

- [ ] **成功有代价，失败有根据**
  - 即使达成目标，角色付出了什么？
  - 失败的原因合理，不是随机降临的倒霉

- [ ] **坚持/放弃符合角色性格与处境**
  - 角色继续尝试的动力有性格、情感、处境三重支撑
  - 多次失败后仍坚持，身体或心理代价写出来了

- [ ] **节奏平衡（连续沉重章节时检查）**
  - 此前2章以上均为压制/失败时，本章是否有反转/笑点/爽点？
  - 笑点/喜剧细节自然生长于人物性格，不是硬插

- [ ] **爽点有前期铺垫**
  - 胜利/突破是之前积累的兑现，不是机械降神

- [ ] **成长、成名与武力进阶有过程**
  - 名声不能一夜跳成“全城皆知”，除非本章核心就是重大公共事件且后续承担代价；常规艺名、花魁名、江湖名号应经历“小圈子注意→稳定表现→关键人物认可→外部传闻→麻烦上门”的阶梯。
  - 技艺不能从“会”直接跳到“三绝/宗师/顶尖”：至少写出旧技法不适用、改学新规矩、失败或别扭、旁人指出、重复练习、一次小场景兑现。
  - 武力不能突然变高：正式秘籍、师承、身体训练、失败记录、时间跨度、对手强弱都要写清。牛阳这类“低存在感/无影步”成长，前期要先写重心、脚步、呼吸、少一步、避开视线，而不是直接写“没人看见他”。
  - 关系不能突然亲密：雇主到亲人、同伴到知己，都要有照拂、试探、误会、共同承担、沉默默契等中间台阶。
  - 检查后续章节是否吃了前文没写过的成果；若某章需要角色“已经成名/已经练成/已经亲近”，前文必须补过程，或把该成果降级为“刚有苗头”。
  - 大纲标签也要守住进阶边界：章节标题旁的小括注、概要词、武力表、笑点表不能提前写“针法初显/杀人剑初成/正式练功”等已经兑现的成果；正式资源未到手前，一律降级为“根苗、本能、旧技、试探、预备动作”。
  - 时间跨度要按角色线分别核对：全书“十五年后”不等于每条关系都经历十五年；入宫五年、从军十五年、练功四年、寻访十年要各算各的。凡出现“七年/十五年里/当年/数年”这类时间词，回到章节节点和人物年龄表核一次。

- [ ] **关键资源递送有可信链路**
  - 秘籍、神兵、信件、药方、身份文书等关键资源不能由神秘人物突然当面塞给主角；必须有信息来源、转交理由、可信中间人和接收者的选择过程。
  - 资源最好从“熟悉之人”手里来：牛阳适合由老乞儿递书，明珠适合由柳七娘递女红图谱，常心适合由佛子递信。中间人要有资格、有关系、有动机。
  - 谋士/无命类角色可以布局，但不要显得像作者之手。先写他通过密会、线报、闲聊、观察获得判断依据，再写他让合适的人转交合适的东西。
  - 明面人物和暗面身份要分工清楚：张佑仕这类明面身份可听曲、看舞、闲聊、试人，不宜当场掏出秘籍；无命这类暗面身份应通过旧档、货路、线人、熟人递送，让资源像从世界里流出来，而不是从作者手里掉下来。
  - 递送前必须有“消息汇流”场面或等价证据：洛阳情况、老乞儿线、西域/边境变化、常心/小天竺寺、北边战事、朝中局势等相关线索，至少写出几项能支撑判断的来源。若某线索只是传闻（如客栈后厨来了个古怪伙计），只作传闻层面记录，不提前点破真实身份。
  - 谋士组织不能只写“众人汇报、主脑下令”。要写出组织器官感：谁负责眼耳（脚印、人流、城门）、谁负责货路（金钱、价码、异常）、谁负责档案（旧纸、库房、抄录）、谁负责场面人情（手腕、眼神、客人分寸）。主脑要把这些触感连成判断，而不是凭空知道。
  - 张佑仕/无命这类核心谋士的资源调度要有“长期努力”支撑：过目不忘、默写典籍/功法、建立库房、秀才分类保管、删去危险门派名、留下发力/步位/呼吸/收势等可用骨架。不要让秘籍像现成道具临时出现。
  - 递送资源前，先写人物分岔判断：这颗种子可能长成什么？会折、会偏、会死心、会被利用，还是会成网、成刀、成路？资源选择应针对“快速成长但不替他选择”的方向。
  - 资源索引要具体：从哪类典籍/残卷中取哪几章，压下哪几章，为什么此时不给全本。给浅不够、给深会害人，这种取舍最能体现谋士分寸。
  - 递送过程要有物理链路：谁抄录、谁转运、藏在哪里、由谁保管、为什么暂压不发。可用“旧档残本→货箱→熟人手边→择时递出”的层次。
  - 接收者不能立刻无条件接受。根据性格写拒绝、迟疑、试探、压下不发、想清楚后回来拿等步骤；这一步是人物自主性的关键。
  - 检查“某人给某人秘籍/宝物/机会”时，问四句：他怎么知道需要给？为什么通过这个人给？接收者为什么会收？收下后付出什么代价或承担什么约束？
  - 传闻只传闻，不代作者下结论：线报、闲聊、旁人转述只能写出“听说”“看着像”的层次，不要让角色一句话就替作者把真实身份、门派出身、全部能力一次解析完。若要埋身份线，只写可见行为、器物、习惯和结果，保留误差和猜测空间，等主角后续亲眼验证。

## 文风统一检查

详见 [writing-style-guide.md](writing-style-guide.md)

- [ ] **叙述语言符合主文风**
  - 古风：无现代口语、称谓准确、心理描写不超三层
  - 严肃/黑暗：无喜剧元素、无煽情总结
  - 幽默/搞笑：笑点未被解释、从情境生长
  - 开朗活泼：节奏轻快、无沉重独白
  - 白话文：无文言腔、对话口语自然

- [ ] **对话语言与主文风一致**
  - 古风文中角色不说"好吧/算了/逻辑"
  - 严肃文中角色不突然搞笑
  - 幽默文中叙述不突然沉重说教

- [ ] **主文风与辅文风比例恰当**
  - 辅文风只在特定角色/场景中出现，不喧宾夺主

## 对话声音一致性检查

详见 [dialogue-consistency.md](dialogue-consistency.md)

- [ ] **遮住角色名，能判断出是谁说的吗？**
  - 判断不出 = 声音不独特或出现了污染

- [ ] **无跨角色污染**
  - A角色没有说出B角色才会说的话（句式/用词/思维方式）

- [ ] **无情绪污染**
  - 逃命/极度恐惧/愤怒状态下，角色没有说出冷静分析的长句

- [ ] **无作者污染**
  - 角色没有替作者做主题发言或背景注释

- [ ] **角色声音与声音档案（Voice Card）一致**
  - 句子长短、词汇层次符合人物档案设定

## 展示而非讲述检查

### 常见"讲述"标记

检查并修正以下模式：

| 讲述（避免） | 展示（使用） |
|-------------|-------------|
| 他很愤怒 | 他握紧拳头，指节发白 |
| 她很美丽 | 他凝视着她，忘记说话 |
| 他很紧张 | 他反复调整领带 |
| 房间很乱 | 衣服扔在沙发上，外卖盒堆在桌上 |
| 他很富有 | 他从口袋里掏出一叠现金 |

### 自查问题

- [ ] 是否直接陈述情绪？（改为身体反应）
- [ ] 是否用形容词总结？（改为具体描写）
- [ ] 是否跳过了关键场景？（补充展示）

## 节奏检查

- [ ] **句子长度有变化**
  - 没有连续 3 句长度相同
  - 长短交错

- [ ] **段落长度适当**
  - 避免大段文字墙
  - 动作场景用短段落

- [ ] **信息密度有变化**
  - 高密度（动作/对话）
  - 低密度（描写/内心）

## 语言检查

- [ ] **没有 AI 写作痕迹**
  - 避免"此外""然而""强调"等 AI 词汇
  - 避免四字成语堆砌
  - 句式多样化

- [ ] **"的"字不密集**
  - 没有连续多个"的"
  - 简化修饰结构

- [ ] **用词精确**
  - 避免模糊词（"一些""某种"）
  - 使用具体词汇
  - 检查“像A又不像A/像A又不像B/像没有/像真听进去/像真事”这类模糊判断句。若只是作者在替读者概括，应改成更准的词或可见细节：可用“煞有介事”“似笑非笑”等成语，也可落到嘴角、眼底、眼皮、手势、脚步、腔调、停顿等动作；不要让读者停下来猜“到底像什么”。

- [ ] **避免机械对仗和空泛警句**
  - 检查“A会B，C也会D”“X在这里，也会被Y”“就算/即便X，也会Y”“不是A，是B”“越A越B”等过于工整的句式；若像作者造句，不像角色说话，必须重写
  - 检查“前句已经下令/表态，后句再补一个对称尾巴”的假余味；若后句没有新信息、新动作或新转折，删掉
  - 例如“挡路的杀。不挡路的，留给路。”后半句只是把“不杀”的意思换壳重说，后文若已有解释，应删到“挡路的杀”
  - 古风武侠可以有短句和留白，但不要为了“有味道”硬造对称句；优先使用角色身份中自然会说的俗语、行话、动作和具体物件
  - 表达“不保险/迟早暴露”时，不写“墙会倒，名字也会漏风”“躲在这里，也会被翻出来”这类模板句；可改为“隔墙有耳，纸包不住火”“藏到废井底下，官府真要翻，也翻得到”等符合场景的说法
  - 改写时先问：这句话若由旁人复述，是否像这个角色的口吻？是否贴合他此刻看见的东西？
  - **动词与对象匹配**：不要把一个好用动词套到所有对象上。屏风本就遮面，评价屏后副琴应写“收得稳/隔一层/不抢声”，不写“藏得深”；纸人不会“藏”，活人面对纸人应写“装、扮、摆、僵、假”，不写“藏得像纸人”；水在草尖是“挂、凝、沾、滚”，不是“藏”。每次用“藏、躲、漏、压、收、挂、浮”等动词，先问这个对象真的能这样动吗？
  - **修辞对象校验**：比喻要保留对象本身的物性。声音可“压、收、托、接、散”，人可“躲、避、扮、装”，物可“藏、露、挂、沉”；若比喻让对象做不了的动作，读者会先出戏。
  - **似是而非句校验**：凡出现“像笑又不像笑”“像看棋又不像”“像谁都不像”等句式，先判断它是否提供了新信息。若只是含混气氛，改为成语、神态或动作；若要保留含混感，必须让含混来自角色感知不足，而不是作者词穷。
  - **机械句改写工具箱**：
    - 换语法：把抽象判断改成因果、转折、递进、设问、半句留白或动作后置；例如“躲在这里，也会被翻出来”改为“藏到废井底下，官府真要翻，也翻得到”
    - 落物件：让句子贴住眼前物件、伤口、地形、器具、天气；先写角色看见/摸到/听见什么，再让判断从细节里出来
    - 用俗语：角色身份允许时，用“隔墙有耳”“纸包不住火”“夜长梦多”“树倒猢狲散”等自然俗语；须符合时代、地域和身份
    - 用行话：军人说军阵、刀口、粮道；商人说账、货、路；僧人说灯、粥、经、因果；牧人说羊、草、水、风
    - 用比喻/隐喻/拟人：比喻必须来自角色生活经验和当前场景，不用漂亮但离题的修辞；一个重场景通常只留一个主比喻
    - 用成语/典故/典籍：只给读书人、僧人、官员、谋士等有知识来源的角色使用；引用须准确，宁可不用，不可张冠李戴或炫技
    - 用沉默替代：若角色不该说透，让动作、停顿、避开目光、收紧手指代替结论
    - 终检：改完后读出声，若句子像作者在写金句，而不是角色在当下说话，再压回白话或动作

- [ ] **深层话术与角色匹配**
  - 需要表达身份遮掩、目的隐藏、明暗互补、权谋试探等复杂意思时，先判断角色是否有资格说深话：僧人、谋士、官员、老江湖、长期逃亡者可以用映射；孩童、普通商客、惊惧中的逃难者通常只说眼前事
  - 深层话术要有两层：表层能被当场角色听懂，暗层能让读者回味；若只有暗层，读者会迷；若只有表层，就过直
  - 避免直白解释“身份/目的/伪装/看清自己”等抽象词，优先用光影、棋局、账本、刀鞘、僧灯、路、影子等角色可感之物映射
  - 表达“明处反而能藏人”时，可用“日头底下也有影子。人人看得见，未必人人低头看”这类话术；不要直接说“他在明处也在暗处，用阴阳互补掩饰身份目的”
  - 判断是否保留深话：删掉暗层后，表层是否仍符合当前对话？若不符合，说明作者意图压过了角色真实；改回具体动作或白话
  - 同一场景避免连续多句深话；一句落下后，让动作、沉默或环境承接

- [ ] **审美描写与场景行当匹配**
  - 描写容貌、风姿、笑意、台上气质时，先判断场景行当：花船、宫廷、文会、江湖、集市的审美语言不同，不可混用。
  - 花船/歌舞/宴席场景不把人脸直接写成“货、价钱、买不起、值钱”等粗买卖口吻，除非刻意表现恶客羞辱；班主、琴师、歌女应更多使用“台、曲、灯、帘、花、月、香、袖、眼波、身段”等行当物件与审美语汇。
  - 可用典雅比喻和成语，但须贴合人物文化层次：柳七娘、琴师可说“隔帘看花”“月隔云好看”“眉眼入曲”；读书人可引“花非花，雾非雾”；粗客可说“秀色可餐”，但“倾国倾城、闭月羞花”要少用，避免套话和堆砌。
  - 直白外貌标签（美、漂亮、好看、贵、媚）要落成可见细节：灯下袖口、眼尾收放、笑意到不到眼底、背脊松紧、指尖停顿、客人举杯忘饮。
  - 若必须写“被物化”的压迫感，用视角承受者的身体反应与环境压力承接，而不是让正面角色也用粗鄙估价语言；恶客说得粗，叙述与可信角色要有更高的审美层次。

- [ ] **武侠古风文风**（历史架空/武侠类型适用）
  - 心理描写点到为止，不做三重并列分析
  - 不解释情绪的来源和机制
  - 用动作/神态代替心理陈述
  - 留白给读者，写七分留三分
  - 详见 [natural-dialogue-techniques.md](natural-dialogue-techniques.md) 武侠古风文风原则

## 最后一步：读者视角终检（最终兜底）

> 在完成字数、连贯性、文风、人物声音等技能检查后，必须再把自己当成第一次读到本章的普通读者，通读一遍。此步是最终兜底，不看作者意图，不替作者补脑，只看读者当下能不能读顺、信服、入戏。凡是读者可能卡住、出戏、疑惑、无感、跳读、觉得作者在解释的地方，都要回到正文微调。
>
> 终检原则：读者只能读到纸面上的字，读不到作者脑子里的设定。纸面上没有铺垫的聪明、深情、痛苦、谋算、成长、身份、关系，都不能算成立。

### 1. 第一口气检查

- [ ] 前 3 段是否让读者立刻知道“出事了”或“有事要发生”？
- [ ] 前 20% 是否有明确冲突、阻力或悬念，而不是单纯铺陈？
- [ ] 读者是否能在开头迅速抓住主角是谁、他此刻想要什么？

### 2. 人物真实感检查

- [ ] 主角的每个重要选择，读者是否能从他的处境、年龄、经验和性格里相信？
- [ ] 人物有没有用行动暴露性格，而不是靠旁白说明性格？
- [ ] 配角出场是否有自己的气味、动作或说话方式，不像临时工具人？
- [ ] 人物没有为作者解释设定、推动剧情而说不该说的话？

### 3. 对话可听性检查

- [ ] 遮住说话人名字，主要角色的对话还能不能分辨是谁？
- [ ] 每句对话是否有目的：推进事件、制造冲突、露出性格、埋伏笔、形成笑点之一？
- [ ] 对话是否留有潜台词，没有把人物心思全讲明？
- [ ] 幽默是否来自人物关系和情境，不靠作者解释笑点？
- [ ] 对话是否符合关系距离：陌生人不审讯式短问，弱势者不无故逼问，上位者不随便解释底牌，熟人之间才可省称谓、省缓冲。
- [ ] 对话是否符合场景压力：逃命时不长篇分析，密会时不说废话，花船场面有行当分寸，军阵场面有军中口吻，佛门场面有慈悲和克制。
- [ ] 对话是否像角色自己要说的话，而不是作者为了递信息硬塞给他；若删掉这句话只少了设定、不少人物反应，就重写成动作、线报、旁人误解或后续验证。

### 4. 语言纸面检查

- [ ] **形容词**：有没有“很美、很冷、很厉害、复杂、深沉、锋利、特别、古怪”等空泛总结？能落成灯光、手势、声音、衣角、伤口、器物反应的，必须落到可见细节。
- [ ] **比喻**：比喻是否来自角色经验和当前场景？牧人用羊、草、水、风；军人用阵、粮道、刀口；僧人用灯、粥、经、因果；花船用台、曲、袖、帘、月、香。漂亮但离场的比喻要删。
- [ ] **比喻物性**：对象能不能做这个动作？声音可“收、托、压、散”，人可“躲、避、装、扮”，水可“挂、凝、滚”，屏风后的琴声不写“藏得深”，活人装纸人不写“藏得像纸人”。
- [ ] **量化**：数字、时间、距离、程度是否让读者更清楚？若“三分、七成、十丈、十五年、半息”没有角色来源或感官依据，就改成可感表达；若涉及年龄线、练功线、路程线，必须回大纲核对。
- [ ] **用词领域**：物品、行当、武功用词是否属于它自己的世界？刀剑写刃口、剑脊、护手；布料写纹路、针脚、洗痕；账房写价码、账面、货路；不要把跨领域词硬套。
- [ ] **现代感**：有没有现代网络腔、心理学腔、作者论文腔混进古风武侠？如“逻辑、情绪价值、身份认同、系统性、复杂情感”等，除非叙事设定允许，否则换成角色时代能说的话。

### 5. 情绪与性格检查

- [ ] 情绪是否靠身体、动作、停顿、选择呈现，而不是直接说“他愤怒/她害怕/心情复杂”？重情绪场面至少有一个可见细节：手、喉、呼吸、衣袖、步子、眼神停留处。
- [ ] 情绪强度是否符合经历重量？灭门、亡国、受辱、断臂、亲近者死亡后，人物不能立刻恢复轻松；若轻松，必须是保护色或压抑，并有破绽。
- [ ] 性格是否前后一致：牛阳少话但会做事，明珠自尊强但会学着收，张佑仕废话是壳、无命才是骨，常心压火不乱烧；每句关键话都要符合其性格底色。
- [ ] 人物有没有越过自己的认知边界？普通香客不精准识军阵，传闻不点破真实身份，孩子不做老谋深算的判断，惊惧中不讲完整谋略。
- [ ] 深层话术是否有资格说：谋士、僧人、老江湖可说映射；普通商客、孩童、逃难者只说眼前物。深话一句足够，连续深话会像作者金句。

### 6. 情节牵引检查

- [ ] 本章目标是否清楚，读者知道主角在追什么、躲什么、怕什么？
- [ ] 阻力是否真实，成功是否付出代价，失败是否有根据？
- [ ] 转折是否来自已写出的信息，而不是突然安排？
- [ ] 结尾是否自然引向下一章，让读者想知道“接下来会怎样”？

### 7. 场景稳定检查

- [ ] 地点、方向、距离、时间是否让读者能在脑中站稳？
- [ ] 关键物件、伤口、气味、声音等细节是否具体，并服务人物或伏笔？
- [ ] 有没有突然出现但此前没有来源的信息？
- [ ] 有没有过度解释、重复说明、现代词、空泛形容词让读者停下来？
- [ ] 场景中的人是否各有位置和目的？大场面不能只写“人很多、很繁华”，要有摊贩、差役、香客、乞儿、商旅、官眷等各自正在做的事。
- [ ] 环境是否参与情绪和行动？紧张场景里火苗、影子、脚步、门轴、雨声要能增加不安；舒缓场景里细节要能让人喘气，不抢戏。
- [ ] 视角是否稳定？一段内不要忽然从牛阳看到明珠心里，又跳到作者上帝视角解释设定；需要切换时，用动作或段落转场承接。

### 8. 重复与机械表达检查

- [ ] 有没有“前一句说明，后一句反过来又说一遍”的重复？如“他没有拔刀。刀还在鞘里。”若第二句无新信息，删。
- [ ] 有没有“不是A，是B”“他没有做X。做了X就会……”这类解释性结构？能用动作表示，就删解释。
- [ ] 有没有机械对仗、伪金句、空泛警句？如“A会B，C也会D”“X在这里，也会被Y”“越A越B”。若不像角色当场自然会说，改成俗语、行话、动作或具体物件。
- [ ] 有没有“像A又不像A/像A又不像B/像真事/像没有察觉”这类似是而非判断？若读者只能得到“很玄”“很复杂”的空感，改成“似笑非笑、煞有介事”或具体表情动作。
- [ ] 有没有同一段反复使用同一个关键词、动作或句式？如连续“看了看、沉默、点头、很、像是”。保留最有用的一处，其余换成更具体反应或删掉。
- [ ] 有没有作者替读者解释已经能看懂的含义？若读者能从动作明白，就不要再写“这说明……”“他知道……”“她明白……”

### 9. 阅读节奏检查

- [ ] 是否有该快的地方快、该慢的地方慢？
- [ ] 动作、对话、环境、内心是否交替出现，没有连续堆一种写法？
- [ ] 重场景是否有情绪落点，轻场景是否有暗线推进？
- [ ] 是否存在读者可能跳读的段落？若有，删、压缩或换成行动/对话。

### 10. 终检操作法

- [ ] 第一遍默读：只标出卡顿、出戏、重复、太直白的位置，不急着改。
- [ ] 第二遍出声读对话：凡听起来像作者讲设定、像现代人、像审讯短问、像同一个人说话的，都改。
- [ ] 第三遍只看形容词和比喻：空泛形容词落细节，错位比喻换成角色生活里的物件。
- [ ] 第四遍只看重复：连续两句表达同意，删后一半；前文已有铺垫，后文不再解释。
- [ ] 第五遍回看改动前后三段，确认节奏、因果、人物声口没有被改坏。

### 11. 微调原则

- 只改会影响读者体验的地方，不为“更华丽”而改。
- 优先修：开头抓力、人物动机、对话声口、结尾钩子、出戏细节。
- 每次调整后重读被改段落前后 3 段，确认节奏和因果没有被改坏。
- 读者视角终检拥有最后否决权：即使前面的文风、逻辑、字数都合格，只要读者视角读起来直白、机械、重复、用词不当、人物不像本人，就必须回修。
