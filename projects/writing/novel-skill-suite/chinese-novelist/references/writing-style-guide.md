# 文风指南与统一检查

## 核心原则

> 文风是小说的底色，不是场景的配乐。场景情绪可以起伏，文风不应跟着切换。确定主文风后，贯穿全书不变；辅文风可以在特定角色或局部场景中使用，但不能喧宾夺主。

---

## 文风选择与定义

创作开始时记录：`主文风 = [选择]` + `辅文风 = [选择或无]`

---

## 八种文风档案

---

### 1. 古风

**适用题材**：武侠、历史架空、剑仙、宫廷

**核心气质**：含蓄克制，精炼留白，动作代替心理，典故代替直白

**语言特征**：
- 短句为主，偶有长句作收尾
- 文白夹杂，但不生硬（"他不言语"而非"他不说话"，但不用"汝""乃"）
- 人称用"他/她"，不用"你"的书面强调
- 称谓：娘/爹/将军/先生，不用"妈/爸"
- 情绪用神态动作表达，不做三重并列分析

**对话特征**：
- 句子短，留白多
- 不解释自己的行为
- 用典故表达观点，不做白话陈述
- 不用现代分析词（"逻辑""系统""心理"）

**禁忌**：
- 心理描写超过三层
- 现代口语（"好吧""算了吧""对啊"）
- 过度排比对仗（显得刻意）
- 现代道德观直接陈述

**示例**：

❌ 错误：
```
他心里有点不安，但同时又有些期待，这种复杂的情绪让他不知道该怎么办。
```

✅ 正确：
```
他站在那里，没动，手指收了收。
```

---

### 2. 古典（文学性）

**适用题材**：历史正剧、文人笔记风、民国背景

**核心气质**：雅正克制，有文学底蕴，用词考究但不晦涩

**语言特征**：
- 长短句结合，重视节奏感
- 可用少量文言词，但以白话为骨架
- 景物描写细腻，情绪隐于意象
- 引用诗词典故须准确，不滥用

**对话特征**：
- 人物说话文雅但不迂腐
- 有礼数，有层次，有回味
- 用典恰当，不炫技

**禁忌**：
- 粗俗口语
- 戏谑调侃（除非是人物性格需要）
- 错用典故或张冠李戴

**示例**：

❌ 错误：
```
"你来啦，快进来坐。"
```

✅ 正确：
```
"来了就好，"他把书搁下，"坐。"
```

---

### 3. 严肃

**适用题材**：战争、悲剧、现实主义、伦理困境

**核心气质**：克制、深沉、不煽情、不幽默，以真实感取胜

**语言特征**：
- 去除所有喜剧性元素
- 情绪压着写，不爆发不宣泄
- 用细节代替感叹
- 节奏慢于动作类，重视停顿

**对话特征**：
- 话不多，但每句有重量
- 不逃避困难话题，但不直接说透
- 沉默和停顿本身是对话的一部分

**禁忌**：
- 喜剧性插曲（除非角色刻意使用来缓解紧张）
- 夸张的情绪表达
- 煽情的总结性语句（"他终于明白了什么是真正的痛苦"）

**示例**：

❌ 错误：
```
泪水夺眶而出，她终于明白了失去的意义。
```

✅ 正确：
```
她没哭，只是站了很久，直到腿有点麻了，才往回走。
```

---

### 4. 幽默

**适用题材**：都市轻喜剧、武侠喜剧、言情喜剧

**核心气质**：轻盈，有落差感，笑点自然生长于情境而非刻意制造

**语言特征**：
- 短句快速切换制造节奏感
- 利用角色性格反差
- 意外的真实（角色说出了不该说的大实话）
- 场景与期待的落差

**对话特征**：
- 角色说话"跑偏"，但在性格范围内
- 答非所问制造喜剧效果
- 不解释笑点（解释即死）
- 严肃人物说出不严肃的话，效果加倍

**禁忌**：
- 解释笑点
- 角色对自己的幽默自我欣赏
- 强行搞笑（笑点不从情境生长）
- 在高情感张力时刻硬插笑点破坏气氛

**示例**：

❌ 错误：
```
"哈哈，你这人真有趣！"他忍不住大笑起来，觉得她说的太好笑了。
```

✅ 正确：
```
"不是说好不说话的吗？"
"我说的是你不说话。"
她噎了一下，"……行。"
```

---

### 5. 搞笑

**适用题材**：喜剧向武侠、穿越喜剧、无厘头题材

**核心气质**：夸张、荒诞、密集笑点，以节奏感和意外取胜

**语言特征**：
- 夸张用词（"惊天地泣鬼神"用于日常小事）
- 荒诞逻辑（从前提到结论的跳跃）
- 密集的意外转折
- 人物内心旁白可以很滑稽

**对话特征**：
- 话多而密，喜剧性打断
- 角色经常误解对方
- 可以有重复性笑点（"老规矩"式的角色特征梗）

**禁忌**：
- 在搞笑文风下使用严肃的情感重场景（会让读者出戏）
- 笑点太稀疏（读者需要密集刺激）
- 重复同一类型的笑点过多

**示例**：

❌ 错误：
```
他很尴尬，感到十分难为情。
```

✅ 正确：
```
他感觉脸上的皮在往地里钻，若不是还要保持面子，他早就把自己埋了。
```

---

### 6. 开朗活泼

**适用题材**：青春校园、轻松冒险、治愈向言情

**核心气质**：轻快、积极、有能量，即使遇到困难也不沉重

**语言特征**：
- 节奏快，短句多
- 感叹、反问用来传递活力
- 比喻轻巧，贴近生活
- 情绪直接但不沉重

**对话特征**：
- 话多，语速快
- 喜欢追问，喜欢反应
- 表达感情直接
- 不掩饰喜好

**禁忌**：
- 沉重的内心独白
- 压抑的景物描写
- 刻意降低调性来"制造深度"（活泼文风的深度来自细节，不来自压抑）

**示例**：

❌ 错误：
```
她站在那里，心里有些复杂，不知道该说什么好。
```

✅ 正确：
```
她拦住他："等等，你刚才说的——你真的是这个意思吗？"
```

---

### 7. 黑暗

**适用题材**：暗黑武侠、悲剧、末世、道德困境

**核心气质**：压抑底色，道德灰色，不提供轻易的救赎或答案

**语言特征**：
- 色调偏冷，意象偏沉
- 细节可以残酷，但不嗜血（残酷要有意义，不是为了震撼）
- 希望要稀薄，不能完全没有（完全绝望=读者放弃阅读）
- 景物描写配合情绪但不煽情

**对话特征**：
- 人物不相信善意
- 说话带刺，或者冷漠
- 直白但不温柔
- 可以有黑色幽默（笑中带苦）

**禁忌**：
- 不必要的残酷细节（为黑暗而黑暗）
- 突然的光明转折（除非是精心铺垫的）
- 正能量说教
- 把黑暗美化

**示例**：

❌ 错误：
```
尽管如此，他心里仍然保留着一丝希望的火花，相信明天会更好。
```

✅ 正确：
```
他把那个念头压下去了。这种东西留着没用，只会让后来的事情更难挨。
```

---

### 8. 白话文

**适用题材**：现代都市、现实主义、民国白话风

**核心气质**：口语化，贴近日常，去除文学腔调

**语言特征**：
- 完全口语化表达
- 无文言成分
- 叙述直接，少修辞
- 对话与生活语言无缝衔接

**对话特征**：
- 就是日常说话，不追求文学感
- 有口头语、填充词
- 可以省略主语
- 重复、打断都是正常的

**禁忌**：
- 文言词或文学腔（"他不言语"这种）
- 过度修辞
- 把日常对话写得太工整

**示例**：

❌ 错误：
```
"你此番前来，所为何事？"
```

✅ 正确：
```
"你来干嘛？"
```

## 混合文风配比

| 主文风 | 辅文风 | 效果 | 注意 |
|-------|-------|------|------|
| 古风 | 幽默 | 武侠喜剧 | 幽默来自人物性格，不破坏古风底色 |
| 严肃 | 黑暗 | 悲剧文学 | 不要加任何喜剧元素 |
| 白话文 | 幽默 | 都市喜剧 | 笑点要接地气 |
| 古典 | 严肃 | 历史正剧 | 保持克制 |
| 开朗活泼 | 幽默 | 青春喜剧 | 控制密度，不要过满 |
| 黑暗 | 幽默 | 黑色幽默 | 幽默必须是苦涩的，不是轻松的 |

## 文风统一检查（每章完成后必查）

### 快速自检三问

1. **这章的语言基调，和上一章一致吗？**
   - 如果不一致，是因为场景需要（如：从战场到温情时刻），还是不小心"滑"出去了？

2. **这章有没有出现不属于主文风的词汇或句式？**
   - 古风文中出现了现代口语？
   - 幽默文中出现了严肃说教？
   - 白话文中出现了文言腔？

3. **对话的风格和叙述的风格是一致的吗？**
   - 叙述是古风，对话却是白话？（角色的口语化需要和文风匹配）

### 逐项检查表

**古风/古典检查：**
- [ ] 无现代口语（妈/爸/老爸/逻辑/心理/系统）
- [ ] 心理描写未超过三层并列
- [ ] 称谓符合时代（娘/爹/将军）
- [ ] 典故引用准确

**严肃/黑暗检查：**
- [ ] 无轻浮喜剧性元素
- [ ] 无煽情总结性语句
- [ ] 情绪克制，不泄露

**幽默/搞笑检查：**
- [ ] 笑点未被解释
- [ ] 笑点从情境/人物性格生长，不是硬插
- [ ] 未在高情感场景中破坏气氛

**开朗活泼检查：**
- [ ] 节奏轻快，无沉重独白
- [ ] 角色情绪直接，不压抑

**白话文检查：**
- [ ] 无文言腔
- [ ] 对话口语自然，无书面痕迹

## 描写句式检查：禁用解释性否定结构

> 这是最常见的AI写作痕迹之一。用否定-肯定或肯定-否定结构解释细节，等于替读者做了感受，破坏了"展示而非讲述"原则。

### 识别方法

凡是出现以下任意结构，必须重写：

**正向否定（先否定再肯定）**
- `不是XX，是XX`
- `那种XX不是XX，是XX`
- `XX不是XX，是XX`

**反向否定（先肯定再否定）**
- `是XX，不是XX`
- `那种XX是XX，不是XX`
- `XX是XX，不是真的XX`

**模糊并列（两个都说不准）**
- `又像是XX，又像是XX`
- `像是XX，也像是XX`

**判断+解释（先给结论再解释）**
- `那种旧是刻意的，不是真的穿旧的——真正穿旧的衣服……`
- `不是因为XX，是因为别的什么`

**程度递进（用比较句堆叠同一状态）**
- `安静了一会儿，比刚才更安静`（同一状态说两遍，第二句没有新信息）
- `冷了，比昨天更冷`（递进本身是废话，直接写冷的具体表现）

**动作+解释（先写动作再解释动作的意义）**
- `他没有念出来。念出来之后，这件事就不只是一笔买卖了。`（解释了沉默的含义，读者不需要被告知）
- `他没有说。说出来就……`（同类结构）

**根本原则**：所有这些结构的共同问题是——作者在替读者做判断，而不是让读者从细节里自己感受。

## 描写句式检查：禁用冗余否定与重复陈述

> 同一件事用两种方式说两遍，第二句没有新信息，必须删掉。古风文中"不是"是现代口语，应替换为"没有"或直接省略否定，只写动作本身。

### 识别方法

**冗余否定（否定句+等价肯定句，说同一件事）**
- `不是拔剑——剑还在鞘里，一寸都没出来`（三句说同一件事）
- `没有出声，嘴里一个字都没有`（两句说同一件事）
- `没有动——他站在原地，脚步没有移动`（重复）

**"不是A是B"的现代口语替换**
- `不是刺，是压` → 直接写动作：`往下一压`
- `不是打脸，是打嘴` → 直接写部位：`打嘴，拳头正面砸进去`
- `不是跑，是走` → 直接写：`走，步子不快`

**修法原则**：
1. 否定句能省则省——直接写发生了什么，不写没发生什么
2. 古风用"没有"代替"不是"：`没有出剑` 而非 `不是拔剑`
3. 动作描写只写一遍，选最精准的那句，删掉等价的重复句

| 原句（冗余） | 问题 | 改后（精简） |
|------------|------|------------|
| `不是拔剑——剑还在鞘里，一寸都没出来` | 三句说同一件事 | `剑鞘动了，没有出剑` |
| `不是刺，是压，顺着骨缝往下一沉` | "不是刺"是废话，直接写压 | `鞘尖点住，顺着骨缝往下一沉` |
| `不是打脸，是打嘴` | 现代口语+冗余否定 | `打嘴，拳头正面砸进去` |
| `没有说话，嘴里一个字都没有` | 两句说同一件事 | `没有说话` |
| `他没有动，站在原地没有移动` | 重复 | `他没有动` |



### 修法：删掉判断，只留具体细节

| 原句（有AI味） | 问题 | 改后（精准直观） |
|--------------|------|----------------|
| 那种旧是刻意的，不是真的穿旧的——真正穿旧的衣服，领口和袖口会先磨损，这件没有 | 先给结论"刻意"，再解释，读者被喂答案 | 衣服洗过很多次，颜色淡了，但领口和袖口没有磨损，针脚还是整的 |
| 不是因为他的脸，是因为别的什么，察合说不清楚 | 先否定一个原因，再给模糊答案，绕了一圈 | 察合说不清楚是什么，只是感觉到了 |
| 眼睛不大，但亮，那种亮不是热情，是一种长期盯着猎物的亮 | "亮"重复三次，解释了读者本该感受到的 | 眼睛不大，但盯着人的时候，对方会不自觉地把话说短 |
| 不是笑，是一种习惯性的表情，像是在算什么 | 先否定再肯定，绕了一圈 | 嘴角总是挂着一点弧度，像是心里有本账，随时在翻 |
| 眼睛半闭着，像是在打瞌睡，又像是在听 | 两个"像是"互相矛盾，模糊掉了 | 眼睛半闭，像一口老井，深不见底 |
| 帐篷里安静了一会儿，比刚才更安静 | 同一状态说两遍，第二句没有新信息 | 帘子放下来。油灯的火苗还在晃，把几个人的影子照得摇摇晃晃，高低不一。 |
| 他没有念出来。念出来之后，这件事就不只是一笔买卖了。 | 解释了沉默的含义，读者不需要被告知 | 那个字，他认识。手指在膝盖上收了一下，又松开，没有说话。 |

### 核心原则

**只写观察者看见的，不写观察者的结论。**

读者看见"领口袖口没有磨损"，自然会想到"这衣服是故意做旧的"——不需要作者说出来。结论一旦说出来，读者的思考就停了。细节留着，读者的感受才是活的。

## 环境描写情绪一致性

> 环境细节不是背景板，是情绪的延伸。场景的情绪状态，应该通过环境细节的选择和描写方式体现，不能用中性/平静的环境描写配紧张/压抑的场景。

### 识别方法

写完环境描写后，问：**这个细节传递的情绪，和当前场景的情绪一致吗？**

| 场景情绪 | 错误的环境描写 | 正确的环境描写 |
|---------|-------------|-------------|
| 紧张/压抑（密谋结束，各怀心事） | 油灯稳住了，影子压在地上，长的长，短的短（平静、静止） | 火苗还在晃，影子摇摇晃晃，高低不一（不稳定、暗示不安） |
| 危险/逼近 | 风吹过来，草动了一下（中性） | 风从草里过来，带着一点什么，不是草的气味（感知异常） |
| 沉默/压力 | 帐篷里很安静（陈述） | 炭火的声音听得见了，之前没有注意到（沉默让细节放大） |

### 修法

1. 确定场景的核心情绪（紧张/压抑/不安/平静/温暖）
2. 选择能承载这种情绪的环境细节（动态/不稳定 → 紧张；静止/温暖 → 安全）
3. 描写方式本身也要配合情绪（紧张场景用短句、动词；舒缓场景可用长句）

## 文风跑偏的常见原因与修法

| 跑偏现象 | 原因 | 修法 |
|---------|------|------|
| 古风文突然出现"好吧/算了" | 对话写顺手后用口语 | 替换为"……罢了""也好" |
| 幽默文突然变说教 | 作者想表达主题 | 把主题转化为情境，让读者自己感受 |
