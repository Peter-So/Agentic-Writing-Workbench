# 对话声音一致性

## 核心原则

> 遮住角色名字，只看对话，你能判断出是谁说的吗？判断不出，说明角色声音不够独特，或者出现了声音污染。

---

## 角色声音档案（Voice Card）

### 建档时机

在**第二阶段规划**时，为每个主要角色建立声音档案，写入 `01-人物档案.md`。

### 声音档案模板

```
## [角色名] - 声音档案

**说话节奏**：[短句/长句/混合]
**词汇层次**：[文雅/口语/粗俗/专业术语]
**常用句式**：[命令式/疑问式/陈述式]
**口头特征**：[填充词、口头禅、特定用词习惯]
**不会说的话**：[某类词汇或句式这个角色不会用]
**情绪表达方式**：[直接说/用动作/沉默/反问]
**示例对话**：
- 愤怒时："[示例]"
- 紧张时："[示例]"
- 轻松时："[示例]"
```

---

### 常见角色类型的声音特征参考

| 角色类型 | 节奏 | 词汇 | 特征 | 不说什么 |
|---------|------|------|------|---------|
| 武将/江湖人 | 短促、有力 | 直白、少修饰 | 说到做到，不解释 | 长篇分析、抒情 |
| 谋士/文人 | 慢、迂回 | 典故、隐喻 | 说话留余地 | 直接表态 |
| 少年/年轻人 | 快、跳跃 | 口语、直接 | 反应大，容易跑偏 | 城府深的话 |
| 权贵/上位者 | 慢、稳 | 模糊、命令式 | 不自我解释 | 求人、示弱 |
| 商人/中间人 | 灵活 | 实用、暗示 | 永远给自己留退路 | 直接表明立场 |
| 黑暗/反派 | 不定 | 精准或冷漠 | 说话有目的，不废话 | 道德说教 |

## 文风与对话声音的匹配

文风不同，同一角色的对话方式也要相应调整。文风决定**整体基调**，角色声音档案决定**个体差异**。

### 各文风下的对话调性

**古风**：
- 叙述者不翻译潜台词
- 角色用典故、比喻表达，不直说
- 沉默和停顿有重量
- 对话短，余韵长

**古典**：
- 用词雅正，有礼数层次
- 角色可以说比较完整的句子
- 情绪藏在措辞里

**严肃**：
- 对话克制，不解释自己的情绪
- 沉默本身是对话
- 不说废话

**幽默**：
- 对话的意外性和反差制造喜剧
- 答非所问、真话歪说
- 不解释笑点，讲完就走

**搞笑**：
- 密集的误解、夸张、打断
- 对话快速来回
- 角色说话可以很蠢但要在性格范围内

**开朗活泼**：
- 直接、热情、反应快
- 问题多，感叹多
- 不拖泥带水

**黑暗**：
- 冷漠或带刺
- 不相信善意，说话有防御性
- 可以有黑色幽默，但苦涩

**白话文**：
- 就是日常说话
- 省略主语、填充词、重复
- 完全口语化

## 声音污染的识别与修复

### 什么是声音污染

一个角色说了另一个角色才会说的话，或者说了超出自身性格/知识/文风的话。

### 五类常见声音污染

**1. 跨角色污染**：A的话听起来像B说的

示例：
```
（牛阳：一个不识字的少年）
"此事关系重大，须得从长计议，不可草率行事。"
```
→ 这是谋士的句式，不是少年的

修法：改回少年语气
```
"等等，这事……别急，我想想。"
```

---

**2. 跨文风污染**：古风文中角色突然说现代话

示例：
```
（古风武侠）"好吧，那就这么说定了。"
```
→ "好吧"是现代口语，古风用"……也罢"或"便这样"

---

**3. 作者污染**：角色说了作者想说的话，不是角色自己会说的

示例：
```
（反派）"我这么做，是因为这个世界本来就是弱肉强食的，
 强者生存，弱者淘汰，这就是天道。"
```
→ 反派在替作者做主题发言，不是真实角色

修法：把"主题"藏进行为，不放进对话

---

**4. 情绪污染**：角色处于激烈情绪时，说出了冷静分析的话

示例：
```
（逃命中）"根据目前情况来看，东边的搜索队更密集，
 西边或许是我们突围的最佳选择。"
```
→ 逃命时没时间这样分析

修法：用短句、碎片、本能
```
"往西！西边！"
```

---

**5. 功能污染**：角色只是为了传达剧情信息而说话，不是在做自己

示例：
```
"你难道忘了，三年前，就是在这里，你父亲牺牲了……"
```
→ 这是在给不知情的读者补背景，不是真实对话

修法：背景信息用叙述传递，对话只说当下

## 每章对话一致性检查（快速版）

每章完成后，逐一核对**出现台词的角色**：

- [ ] **这个角色的句子长短，和声音档案一致吗？**
- [ ] **这个角色的用词层次，和声音档案一致吗？**（不该文雅的没文雅，不该粗俗的没粗俗）
- [ ] **遮住角色名，还能判断出是谁说的吗？**
- [ ] **这章有没有角色在情绪极端时说出过于冷静的分析？**
- [ ] **有没有角色在传达剧情信息而非说自己会说的话？**
- [ ] **对话的语言风格和全书主文风一致吗？**

---

## 跨章节声音一致性检查

每5章做一次完整检查：

1. 从各章随机抽取3段对话，遮住角色名，判断是否能识别说话人
2. 检查主要角色的口头习惯是否在全书中保持一致
3. 确认没有角色因剧情需要而"临时改变性格"说了不该说的话

---

## 角色声音随故事成长的变化

角色的声音可以随故事**成长**，但必须有根据：

- 一个原本话少的角色，随着信任建立，逐渐话多了 → **合理**
- 一个原本口语化的少年，经历大事后说话变得沉稳 → **合理，需要过渡场景**
- 一个角色本章说话古典，下章突然口语化，没有任何剧情变化 → **声音污染**

**记录成长节点**：在 `01-人物档案.md` 中标注角色声音发生变化的章节和原因。
