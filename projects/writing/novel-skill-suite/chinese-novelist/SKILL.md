# Chinese Novelist: 中文小说创作助手

## 核心流程

### 第一阶段：6问确认

**逐一询问用户。Codex 默认直接用简短自然语言提问；如果用户已经给出足够信息，可跳过对应问题并记录推断。**

---

**问题1：题材与风格**

```
Question: 你想要创作什么题材的小说？
Options:
- 悬疑推理（侦探、破案、解谜）
- 现代言情（都市、职场、恋爱）
- 古代言情（宫廷、江湖、穿越）
- 奇幻玄幻（魔法、异世界、修真）
- 科幻未来（科技、太空、末世）
- 武侠仙侠（江湖、门派、飞升）
- 历史架空（朝堂、战争、权谋）
- 都市现实（生活、成长、社会）
```

用户选择后记录：`题材 = [用户选择]`

---

**问题2：文风**

```
Question: 小说的主要文风是？（可选主+辅，如"古风+幽默"）
Options:
- 古风（精炼留白、武侠历史专用，见 writing-style-guide.md）
- 古典（文学性、文白夹杂、雅正克制）
- 严肃（克制深沉、不煽情不幽默）
- 幽默（落差感、笑点自然生长于情境）
- 搞笑（夸张荒诞、密集笑点）
- 开朗活泼（轻快节奏、积极底色）
- 黑暗（压抑底色、道德灰色）
- 白话文（口语化、贴近日常）
```

用户选择后记录：`主文风 = [用户选择]` + `辅文风 = [用户选择或无]`

### 第二阶段：规划 + 二次确认

执行以下步骤：

1. **创建项目文件夹**：`novels/[小说名称]/`
2. **生成大纲**：创建 `00-大纲.md`，使用 `references/outline-template.md` 模板，填入完整的章节规划；**在大纲首部记录：主文风、辅文风**
3. **生成人物档案**：创建 `01-人物档案.md`，使用 `references/character-template.md` 模板，创建主角、反派、配角档案；**为每个主要角色建立声音档案（Voice Card）** → [dialogue-consistency.md](references/dialogue-consistency.md)

完成后，向用户展示规划摘要并请求确认，等待用户确认。用户同意后，进入"第三阶段：疯狂创作"。

---

### 第三阶段：疯狂创作

**重要：全程无需再次向用户确认，必须逐一章创作**

按顺序逐章创作，每章执行完整的创作流程（见下方"逐章创作"），完成一章后自动继续下一章，直到所有章节完成。

## 疯狂创作——逐章创作流程

每章创作时严格执行以下步骤：

#### 1. 写前分析

1. 读取 `00-大纲.md` - 查看TODO list和已完成章节的摘要
2. 读取 `00-大纲.md` 中上一章的摘要
3. **跨章节重复检查**：读取上一章正文，逐一核对本章大纲中的每个场景/事件，确认上一章是否已经写过。若已写过，本章必须跳过或从该事件之后续写，不得重复。→ [cross-chapter-duplicate-check.md](references/cross-chapter-duplicate-check.md)
4. 更新`00-大纲.md` 中 TODO list - 将本章标记为"进行中"
5. 设计开头钩子 - **最关键**：前20%必须有即时冲突 → [chapter-guide.md](references/chapter-guide.md)（10种开头技巧）
6. 规划场景 - 确定本章需要3-5个场景

#### 2. 撰写

6. 创建章节文件 - 使用`references/chapter-template.md` 模板
7. 撰写正文 - **每章通常以4000-5000字为目标**
   - 开头检查：前20%是否极其吸引人？
   - **文风一致性**：本章叙述/对话是否符合主文风？→ [writing-style-guide.md](references/writing-style-guide.md)
   - 对话规范 → [dialogue-writing.md](references/dialogue-writing.md)
   - 自然对话写作技巧 → [natural-dialogue-techniques.md](references/natural-dialogue-techniques.md)
   - 场景结构写作技巧 → [scene-structure-techniques.md](references/scene-structure-techniques.md)
   - **本章有新角色首次出场？** → [first-encounter.md](references/first-encounter.md)（外形/眼神/动作/语气/气味/直觉/埋钩，至少覆盖4-5项）
   - **情节波折原则** → [plot-tension.md](references/plot-tension.md)（一波多折、失败为常态、坚持须有根据）
   - 内容不足？使用 [content-expansion.md](references/content-expansion.md) 扩充技巧
8. 章节结尾：**流畅过渡优先，悬念钩子非强制** → [plot-tension.md](references/plot-tension.md)（章节衔接原则）；如需钩子参考 → [hook-techniques.md](references/hook-techniques.md)
9. **字数检查** - 必须使用脚本检查字数：`python scripts/check_chapter_wordcount.py <章节文件路径>`，按以下规则处理：
   - **缺失10%以上**（低于3600字）：不扩充现有句子，新增关联情节为现有情节补充动机支撑和细节
   - **缺失5%以内**（3600-3800字）：适当增加现有情节的行为动作、场景描述、语气口吻、肢体语言等细节
   - **超出5%以上**（高于5200字）：先判断内容完整性与必要性；若冗余明显，进行第1次精简，删去重复解释、无效动作、无新信息环境描写和可合并对白
   - **第1次精简后仍明显超出**：再判断是否仍有非必要内容；若有，进行第2次精简，只压缩不影响人物弧、关键冲突、伏笔和情绪落点的段落
   - **第2次精简后仍超出**：若章节内容完整、必要且阅读顺畅，允许保留，不再受字数上限限制；不得为了凑数字删掉关键情节、人物动机、打斗因果、情感落点或章节钩子
   - **超出5%以内**（4000-5200字）：合规，无需调整

#### 3. 撰写后优化

10. 连贯性检查 → [consistency.md](references/consistency.md) - 人物一致性、情节连贯、节奏控制
11. **情节波折与节奏检查** → [plot-tension.md](references/plot-tension.md)
    - 主要行动是否有真实阻力？成功是否有代价？
    - 角色的坚持/放弃是否有性格和情感根据？
    - 连续沉重章节是否安排了反转/笑点/爽点透气？
    - 章节结尾是否自然落地？（流畅过渡优先于强制钩子）
12. **文风统一检查** → [writing-style-guide.md](references/writing-style-guide.md)
    - 叙述语言是否符合主文风？有无跑偏词汇/句式？
    - 主文风与辅文风的比例是否恰当？
13. **对话声音一致性检查** → [dialogue-consistency.md](references/dialogue-consistency.md)
    - 遮住角色名，能判断出是谁说的吗？
    - 有无跨角色污染、情绪污染、作者污染？
    - 对话语言风格与主文风是否一致？
14. **角色认知边界检查** → [character-cognitive-authenticity.md](references/character-cognitive-authenticity.md)
    - 角色说出的目的地/方向，在此之前有没有信息来源？
    - 角色的分析判断，是否超出其年龄/经历/当前情绪状态？
    - 空洞动作（眨了眨眼/摇了摇头/点了点头）是否替换为内心活动+表情+动作组合？
    - **立场一致性**：每句重要对话，说话的角色在他自身的利益/立场框架里，有没有理由说这句话？强势角色是否主动暴露了对自己无威胁的目标的价值？
    - **预知模式检查**：全文搜索"不知道为什么""眼睛先知道""说不清楚为什么记住"——凡出现此类句式，检查是否有具体感知来源；同一模式全书只能出现一次；父亲/长辈的教导内容必须是实用经验，不能是神秘化表述
15. **首次出场检查**（本章有新角色首次出现时必查）→ [first-encounter.md](references/first-encounter.md)
    - 外形是通过观察者感知呈现的，还是直接列举？
    - 有没有眼神、动作、语气、气味中至少两项具体细节？
    - 观察者的直觉反应是否模糊而非精准？
    - 首次出场结尾是否留下了至少一个可激活的钩子？
    - **衣着描写**：是否用了"普通/贵重/细/粗"等概述词？替换为观察者能感知的具体材质/纹路/磨损细节
    - **跨文化面貌描写**：观察者是否用了本族审美标准描述异族人？改为感知差异，不做美丑判断
    - **密谋/谈判/对话/辩论/场景**：每个阵营/每个人，的表面目的/深层目的/第三层默契是否都有体现？→ [first-encounter.md](references/first-encounter.md)（群体会议多层次目的）
15. **深度润色（去除AI味）** - 重点检查并修改：
    - **去除过度修饰的形容词**：删减"璀璨"、"瑰丽"、"绚烂"等AI常用词堆砌
    - **减少抽象陈述**：把"心中涌起复杂的情感"改为具体动作/对话
    - **打破四字格律**：避免"心潮澎湃、热血沸腾"等陈词滥调
    - **增加口语化表达**：人物对话要有个性，避免"书面语套话"
    - **优化节奏感**：长句和短句交替，避免句式单调
    - **细节具象化**：用具体的视觉/听觉/嗅觉细节替代笼统描述
    - **禁用解释性否定结构**：凡出现"不是A，是B"/"是A，不是B"/"那种A是B，不是真的B"/"又像是A，又像是B"/"不是因为A，是因为别的"，必须重写——删掉判断结论，只留观察者看见的具体细节 → [writing-style-guide.md](references/writing-style-guide.md)
    - **禁用程度递进废话**：凡出现"XX了，比刚才更XX"/"XX，比之前更XX"，第二句没有新信息，必须删掉，改为写出状态变化的具体表现（声音、动作、环境细节）
    - **禁用动作+解释结构**：凡出现"他没有做X。做了X就会……"/"他没有说。说出来就……"，解释了沉默/克制的含义，读者不需要被告知——删掉解释句，改为写出克制时的身体反应（手指收紧、眼神停顿、沉默的时长）
    - **禁用冗余否定与重复陈述**：同一件事用两种方式说两遍，第二句没有新信息，必须删掉。识别方法：否定句后紧跟等价肯定句（"不是拔剑——剑还在鞘里"）；动作描写出现两个意思相同的句子。古风文中"不是A是B"须替换为"没有A"或直接只写B的动作 → [writing-style-guide.md](references/writing-style-guide.md)
    - **禁用情绪标签代替身体细节**：凡出现"眼神有劲/憋着/等着/锐利"等抽象情绪标签，必须拆解为至少两个不同部位的可见细节叠加（身子前倾+指节压白+嘴唇收着），不能只靠眼神单一描述 → [character-cognitive-authenticity.md](references/character-cognitive-authenticity.md)
    - **环境情绪一致性**：写完环境描写后检查——这个细节传递的情绪，和当前场景一致吗？紧张/压抑场景禁用平静/静止的环境描写，改为不稳定/动态细节（火苗晃动、影子摇晃）→ [writing-style-guide.md](references/writing-style-guide.md)
    - **高位者视角**：将领/首领/谋士按类别和行动思考，不按个人目标思考——凡出现高位角色点名具体目标（"那个公主，我来处理"），检查该角色是否有个人动机（仇恨/执念/命令），没有则改为行动类别表述（"收尾的事我来"）→ [character-cognitive-authenticity.md](references/character-cognitive-authenticity.md)
    - **禁用笼统物品描述**：凡出现"是旧的""是新的""是好的""不显眼"等概述词，必须替换为观察者能感知的具体细节——材质（牛皮/黑漆木/铜）、纹路（回纹/阴刻）、磨损状态（边角起毛边/漆面裂纹/皮绳颜色深）、尺寸比例（两指宽/比前臂短一截）。武器描写须覆盖：剑鞘材质与磨损、护手形制、剑柄缠法与颜色、有无剑穗；衣物描写须覆盖：织法可见特征、洗涤痕迹、针脚与磨损部位；器物描写须覆盖：材质、尺寸、纹饰、使用痕迹。
    - **物品描写用词须符合领域**：描写物品时，用词必须属于该物品的专属语境，不能跨领域借用——布料用"横竖线""纹路""织法"，不用"经纬"（经纬是地理方位词）；刀剑用"刃口""剑脊""护手""剑格"，不用建筑或地理词汇；尺寸用"指宽""寸""分"等可感单位，不用抽象比较。写完物品描写后逐词检查：这个词，在这个物品的工匠/使用者口中，说得出来吗？说不出来就换。
16. **写作质量深度检查**（见下方"写作检查方法"完整清单）
17. **字数检查** - 必须使用脚本检查字数：`python scripts/check_chapter_wordcount.py <章节文件路径>`，按以下规则处理：
    - **缺失10%以上**（低于3600字）：不扩充现有句子，新增关联情节为现有情节补充动机支撑和细节
    - **缺失5%以内**（3600-3800字）：适当增加现有情节的行为动作、场景描述、语气口吻、肢体语言等细节
    - **超出5%以上**（高于5200字）：先判断内容完整性与必要性；若冗余明显，进行第1次精简，删去重复解释、无效动作、无新信息环境描写和可合并对白
    - **第1次精简后仍明显超出**：再判断是否仍有非必要内容；若有，进行第2次精简，只压缩不影响人物弧、关键冲突、伏笔和情绪落点的段落
    - **第2次精简后仍超出**：若章节内容完整、必要且阅读顺畅，允许保留，不再受字数上限限制；不得为了凑数字删掉关键情节、人物动机、打斗因果、情感落点或章节钩子
    - **超出5%以内**（4000-5200字）：合规，无需调整

#### 4. 最后收尾

16. 生成章节摘要 - 在 `00-大纲.md` 添加摘要（300-500字）
17. 更新状态 - `00-大纲.md` 中 TODO list - 将本章标记为"完成"

## 四大黄金法则

1. **展示而非讲述** - 用动作和对话表现，不要直接陈述
2. **冲突驱动剧情** - 每章必须有冲突或转折；失败是常态，坚持须有角色根据
3. **节奏张弛有度** - 连续压制后给爽点；章节衔接流畅，钩子非强制
4. **文风声音统一** - 全书主文风贯穿不变；每个角色声音独特且前后一致

### 字数检查脚本

使用 `scripts/check_chapter_wordcount.py` 检查章节字数：

```bash
# 检查单个章节
python scripts/check_chapter_wordcount.py novels/小说名/第01章.md

# 检查所有章节
python scripts/check_chapter_wordcount.py --all novels/小说名/

# 自定义最小字数
python scripts/check_chapter_wordcount.py novels/小说名/第01章.md 4000
```

**字数调整规则**（目标区间 4000-5000 字；上限服从内容完整性）：

| 实际字数 | 偏差 | 处理方式 |
|---------|------|---------|
| 低于 3600 字 | 缺失 10%+ | 新增关联情节，为现有情节补充动机支撑和细节；**不扩充现有句子** |
| 3600-3800 字 | 缺失 5% 以内 | 适当增加现有情节的行为动作、场景描述、语气口吻、肢体语言等细节 |
| 4000-5200 字 | 超出 5% 以内 | **合规，无需调整** |
| 5200 字以上 | 超出 5%+ | 先判断内容完整性与必要性；如有冗余，进行第1次精简，删重复解释、无效动作、无新信息环境描写、可合并对白 |
| 第1次精简后仍明显超出 | 大幅超限复检 | 再判断是否仍有非必要内容；如有，进行第2次精简，只压缩不影响人物弧、关键冲突、伏笔、打斗因果和情绪落点的段落 |
| 第2次精简后仍超出 | 内容必要 | 若章节完整、必要且阅读顺畅，允许保留，不再受字数上限限制；不得为了数字牺牲关键情节 |

## 写作检查方法

> 出版社编辑视角 × 金庸/古龙风格标准。每章完成后逐项过检，发现问题立即修改，不积压到全书完成后。

---

### 一、开篇钩子检查

**核心问题：前300字能不能让人放不下？**

| 检查项 | 合格标准 | 常见问题 |
|--------|----------|----------|
| 即时冲突 | 前20%出现真实张力或悬念 | 大段背景铺垫，读者还没入戏就睡着了 |
| 感官锚点 | 有具体的视觉/听觉/触觉细节 | 开篇全是抽象叙述，没有画面感 |
| 人物在场 | 主角或关键人物立刻出现 | 先写天气写环境，人物迟迟不露面 |
| 语气定调 | 第一句话的语气与全书文风一致 | 开头文风和后文割裂，像两个人写的 |

**自检方式**：把开头300字单独拿出来，问自己：如果这是书店里随手翻开的一页，我会继续读吗？

---

### 二、人物声音检查

**核心问题：遮住名字，还认得出是谁说的吗？**

#### 2.1 对话辨识度
- 每个主要角色有没有专属的口头禅、句式习惯、用词偏好？
- 同一场景里，不同角色的对话是否有明显区分，还是都像同一个人在说话？
- 角色的说话方式是否随情绪变化？（平静时和愤怒时的句式应该不同）

#### 2.2 声音污染排查
- **跨角色污染**：A角色的口头禅出现在B角色嘴里
- **情绪污染**：角色刚经历了重大打击，下一句对话却轻描淡写，情绪断层
- **作者污染**：角色说出了超出其认知/身份/时代的话（农村老汉说出了现代网络用语；古代武夫说出了心理学术语）
- **身份污染**：贩夫走卒说话像读书人，江湖草莽说话像朝廷官员

#### 2.3 武侠/古典风格专项
- 江湖人物的说话方式是否有江湖气？（不是文绉绉，也不是现代白话）
- 文白夹杂的比例是否自然？（不能一句文言接一句现代口语，割裂感强）
- 武功招式、门派称谓、江湖规矩的用词是否前后统一？

---

### 三、情节逻辑检查

**核心问题：每一步发展，读者信吗？**

#### 3.1 动机链完整性
- 角色做每一个重要决定，前文是否有足够的动机铺垫？
- 突然的转变（突然信任、突然背叛、突然放弃）是否有情感或事件支撑？
- 反派的行动是否有自己的逻辑，还是只是为了给主角制造麻烦？

#### 3.2 信息对等检查
- 角色知道的信息，读者是否也知道了？（避免角色突然"想通了"但读者不知道他想通了什么）
- 读者知道的信息，角色是否不该知道？（避免角色提前知道还没发生的事）
- 伏笔是否在合理的时间内照应？（埋了伏笔却忘了收，或者收得太快太刻意）

#### 3.3 因果链检查
- A发生→B发生→C发生，每一步的因果是否清晰？
- 有没有"为了情节需要"而强行发生的事？（主角恰好在场、敌人恰好犯蠢、巧合叠巧合）
- 高潮事件的爆发，前文是否有足够的积累？（不能无中生有，也不能积累过长让读者等得不耐烦）

---

### 四、节奏张力检查

**核心问题：读者的心跳有没有跟着故事起伏？**

#### 4.1 章节内节奏
- 本章有没有至少一个节奏变化点？（从紧张到舒缓，或从平静到爆发）
- 场景切换是否流畅？有没有突兀的跳跃或拖沓的过渡？
- 对话和叙述的比例是否合适？（全是对话显得轻浮，全是叙述显得沉闷）

#### 4.2 全书节奏（每5章检查一次）
- 连续几章都是高压紧张，有没有安排透气的场景？
- 连续几章都是平铺直叙，有没有安排冲突或转折？
- 爽点、笑点、泪点的分布是否均匀，还是全堆在某几章？

#### 4.3 武侠专项节奏
- 武打场面的描写节奏：是否做到了"快打慢写"？（关键一招用慢镜头，普通交手快速带过）
- 武打场面之后是否有情绪落地？（打完就走，读者的情绪没有出口）
- 江湖行走的过渡章节是否有内容支撑，还是纯粹的交代位移？

---

### 五、细节真实性检查

**核心问题：读者会不会在某个细节上出戏？**

#### 5.1 物理真实性
- 人物的行动在物理上是否可能？（距离、时间、体力是否合理）
- 武功描写是否有内在逻辑？（不能今天一掌打飞山，明天被小兵一刀砍伤）
- 环境描写是否自洽？（白天写的场景，到了晚上还在同一个地方，时间对不上）

#### 5.2 历史/文化真实性（古代/武侠题材）
- 时代背景下的器物、称谓、礼仪是否准确？
- 有没有出现明显的时代错误？（唐朝人用明朝的官职称谓；宋朝人喝茶的方式是清朝的）
- 江湖规矩、门派礼仪是否前后一致？

#### 5.3 心理真实性
- 角色在极端情境下的反应是否符合人性？（不能太英雄主义，也不能太软弱无能）
- 角色的成长是否有足够的事件支撑？（不能突然就"想通了"、"变强了"）
- 创伤和失去的重量是否被认真对待？（重要人物死亡后，主角不能立刻若无其事）

---

### 六、文字质量检查

**核心问题：每一句话都是必要的吗？**

#### 6.1 冗余排查
- 有没有重复表达同一个意思的句子？（说了一遍，又换个方式说了一遍）
- 有没有可以删掉、不影响理解的段落？
- 有没有过度解释读者已经能理解的内容？（"他很愤怒，因为他感到非常生气"）

#### 6.2 句式多样性
- 连续三句以上是否都是同一句式结构？（主谓宾/主谓宾/主谓宾，读起来像机器人）
- 长句和短句是否交替出现？（紧张场面多用短句，舒缓场面可用长句）
- 段落长度是否有变化？（全是长段或全是短段，视觉上都很疲劳）

#### 6.3 用词精准性
- 有没有用了"差不多对"的词，但有更准确的选择？
- 有没有反复出现的高频词？（同一页里"突然"出现五次）
- 形容词是否具体？（"很大的声音"→"震得窗纸颤抖的声音"）

#### 6.4 古风/武侠文字专项
- 有没有现代网络用语混入古风叙述？（"整个人都不好了"、"画风突变"）
- 有没有过度堆砌文言词汇，反而读不通顺？
- 武功招式名称是否有画面感，还是只是随机组合的汉字？

---

### 七、情感共鸣检查

**核心问题：读者会不会在某个地方停下来，心里有点什么？**

#### 7.1 情感落点
- 本章有没有一个让读者情绪真正被触动的时刻？
- 那个时刻是通过具体细节写出来的，还是靠直接陈述"他很悲伤"？
- 情感高潮之前，是否有足够的铺垫让读者投入？

#### 7.2 人物温度
- 主角有没有让读者觉得"这个人是真实的"的时刻？（不完美、有弱点、有执念）
- 配角有没有自己的生命感，还是只是功能性的存在？
- 反派有没有让读者理解（不一定认同）其行为逻辑的时刻？

#### 7.3 留白与克制
- 有没有本该留白的地方，却写得太满、太直白？（最动人的时刻往往不需要解释）
- 有没有本该点破的地方，却过度留白，让读者困惑？
- 结尾是否给读者留了回味的空间，还是把所有意思都说尽了？

---

### 八、伏笔与照应检查

**核心问题：埋下去的，都收回来了吗？**

#### 8.1 伏笔清单（每10章更新一次）

在 `00-大纲.md` 维护一个伏笔追踪表：

```
| 伏笔内容 | 埋入章节 | 预计照应章节 | 是否已照应 |
|----------|----------|--------------|------------|
| 玉环的来历 | 第05章 | 第52章 | ✅ |
| 碱湖的秘密 | 第01章 | 第44章 | ✅ |
| 神医的身份 | 第39章 | 未定 | ⏳ |
```

#### 8.2 照应质量
- 伏笔照应时，读者能感受到"原来如此"的惊喜感吗？
- 照应是否太刻意，让读者觉得是强行圆场？
- 重要伏笔的照应是否给了足够的篇幅，还是一笔带过？

#### 8.3 细节首尾呼应
- 开篇出现的意象/物件/场景，结尾是否有回应？
- 人物弧线的起点和终点是否形成对照？（第一章的他和最后一章的他，变化是否清晰可见）

---

### 九、全书级检查（每完成1/3进行一次）

> 不是逐章检查，而是从高处俯瞰整本书的结构。

#### 9.1 主题一致性
- 全书想说的那件事，在每一条主线里都有体现吗？
- 有没有某条支线游离于主题之外，读完让人觉得"这段写了干什么"？
- 结局是否回应了开篇提出的核心问题？

#### 9.2 人物弧线完整性
- 每个主要角色的成长弧线是否完整？（有起点、有转折、有终点）
- 角色的变化是否有足够的事件支撑，还是突然就变了？
- 配角的命运是否得到了交代？（不需要每个人都有完整结局，但不能凭空消失）

#### 9.3 结构平衡性
- 三幕结构（建立→对抗→解决）的比例是否合适？（对抗部分通常占全书50-60%）
- 高潮是否在全书的80-90%处？（太早读者意犹未尽，太晚读者已经疲惫）
- 尾声是否简洁有力，还是拖沓冗长？

---

### 十、武侠/古典题材专项检查

> 仅适用于武侠、仙侠、古代言情、历史架空等题材。

#### 10.1 江湖气韵
- 江湖的规矩、人情、恩怨是否有自己的逻辑体系？
- 江湖人物的行事方式是否符合江湖逻辑，还是用现代人的思维在古代江湖里行走？
- 江湖的残酷性是否被认真对待？（不能只有侠义，没有代价）

#### 10.2 武功体系自洽
- 全书的武功强弱体系是否前后一致？
- 武功的习得是否有合理的时间和代价？（不能今天开始练，明天就天下无敌）
- 武打描写是否有画面感，还是只是招式名称的堆砌？

#### 10.3 金庸/古龙风格对照
**金庸风格检查**：
- 人物是否有历史厚度和时代背景的支撑？
- 武功是否有文化渊源（儒释道、诗词典故）？
- 情节是否有家国情怀的底色？

**古龙风格检查**：
- 对话是否简洁有力，每句话都有重量？
- 悬念是否通过留白和节奏制造，而非靠信息堆砌？
- 人物是否有孤独感和宿命感的底色？

#### 10.4 打斗场面专项检查

> 每场打斗都是人物关系、实力对比、心理博弈的集中爆发。招式名称是最低层次的描写，读者要感受到的是力量、危险、智慧、和人在极限状态下的真实反应。

**写前必做：打斗要素分析表**

每场打斗开始前，先填写以下分析，再动笔：

| 要素 | 甲方 | 乙方 |
|------|------|------|
| 武力上限（能打到多强） | | |
| 当前状态（受伤/疲惫/情绪） | | |
| 武功特点（刚猛/轻灵/阴毒/防守） | | |
| 内功深浅（决定持久力和招式威力） | | |
| 身法优劣（速度、步法、空间感） | | |
| 心法/心态（冷静/愤怒/恐惧/轻敌） | | |
| 场景利用（地形/光线/障碍物/天气） | | |
| 真实目的（杀死/试探/拖延/逃跑/震慑） | | |

**打斗节奏：快打慢写原则**

- 普通交手：一句话带过，不铺陈
- 关键一招：慢镜头，写力道、角度、身体感受、对方反应
- 转折时刻（被逼到绝境/突然反击）：最慢，写内心+身体+环境三层叠加

**逐项检查清单**

**智力/心理层**
- [ ] 双方是否都在读对方？（不能只有一方在思考，另一方只是挨打）
- [ ] 有没有虚招、诱敌、示弱、反算？（真正的高手打的是对方的判断，不是对方的身体）
- [ ] 心态变化是否写出来了？（从轻敌到重视，从冷静到慌乱，从绝望到破釜沉舟）
- [ ] 攻心时刻：有没有用话语/行动打击对方的心理防线？

**武力/武功层**
- [ ] 每一招是否有具体的身体感受？（不是"一掌击出"，是"掌风带着热气压过来，他感觉到皮肤上的灼烧"）
- [ ] 内功的存在感：气息、劲力、内伤是否有体现？（内功深的人，招式之间有余力；内功浅的人，几招之后气息就乱了）
- [ ] 武功特点是否在打斗中体现？（刚猛的人不会用轻灵的步法；防守型的人不会主动进攻）
- [ ] 受伤是否有重量？（被打中不能若无其事，要写出伤势对后续的影响）

**身法/空间层**
- [ ] 打斗空间是否清晰？（读者能在脑子里画出双方的位置吗？）
- [ ] 身法是否有个性？（每个人移动的方式应该和他的武功风格一致）
- [ ] 场景是否参与了打斗？（地形、障碍物、光线、天气——不能只是背景板）
- [ ] 距离感是否准确？（远距离用暗器/轻功，近身用短打/擒拿，不能混用）

**节奏/情绪层**
- [ ] 打斗有没有起伏？（一边倒的碾压不是好打斗，要有压制→反压制→决出胜负的节奏）
- [ ] 打斗结束后是否有情绪落地？（胜者的状态、败者的反应、旁观者的感受——打完就走是浪费）
- [ ] 这场打斗之后，人物关系/格局有没有变化？（打斗是推动剧情的，不是展示武功的）

**常见问题与修法**

| 问题 | 表现 | 修法 |
|------|------|------|
| 招式堆砌 | 连续列举十几个招式名称，没有感受 | 每场打斗最多写3-5个有名字的招式，其余用感受和结果代替 |
| 双方轮流挨打 | 甲出招→乙接招→乙出招→甲接招，像下棋 | 加入身法、内力、心理的博弈，让打斗有层次 |
| 受伤无重量 | 被重创后继续如常发挥 | 伤势必须影响后续：气息乱了、某个部位不能用力、速度慢了 |
| 场景是背景板 | 打斗在空旷地进行，环境没有参与 | 让地形/障碍/天气成为变量：狭窄空间限制身法，雨天影响脚力 |
| 心理描写缺失 | 只写身体动作，没有内心活动 | 关键时刻写一句内心：不是分析，是本能反应（"他知道这一招躲不过去了"） |
| 高手打低手无悬念 | 碾压式描写，读者无法投入 | 即使实力悬殊，也要写出低手的挣扎和高手的代价（时间、体力、暴露位置） |

#### 10.5 高手碾压式短打：克制展示武功深度

> 适用场景：高手以极小动作制服对手，目的是震慑而非杀伤，全程收着力道。这类打斗的核心不是"打了多久"，而是"用了多少"——越少越深。

**写前必做：克制展示分析表**

| 维度 | 要填写的内容 |
|------|------------|
| 武器用法 | 用刃还是用鞘？用剑还是用拳？选择本身就是信息（用鞘=不屑出剑） |
| 受力部位 | 点在哪里？（腕内侧骨缝/肘关节/膝窝）部位越精准，说明内功越深 |
| 力道控制 | 够废一条手腕，不多不少——写出"刚好"的感觉，不是"随手" |
| 空间位移 | 打飞了多远？压塌了什么？环境参与打斗，不是背景板 |
| 时间压缩 | 几下功夫在一息之内——旁观者还没反应过来，已经结束了 |
| 声音层次 | 先是骨头脱臼的无声，再是拳砸毡布的闷响，最后才是伤者的声音——声音出现的顺序本身是节奏 |
| 收着的信号 | 明确写出"这是收着的"——让读者自己推算出剑会是什么结果 |
| 旁观者反应 | 分层写：①同伴愣了一息才动；②守门士兵等命令不动；③帐篷里的人听见声音各自反应——每一层都是对这个人武力的侧面注脚 |
| 伤者细节 | 血、碎牙、声音模糊——具体到可以想象，不用"惨叫一声" |
| 打完之后 | 胜者步子不停，不低头看——这个动作本身是最后一句话 |

**逐项检查清单（克制展示型打斗）**

- [ ] 武器是否"降级使用"？（用鞘不用刃，用拳不用剑——降级本身说明实力）
- [ ] 受力部位是否精准到骨骼/关节层面？（"手腕内侧骨缝"比"手腕"信息量大十倍）
- [ ] 力道是否写出了"刚好够"的控制感？（不是随手，是精确）
- [ ] 空间位移是否有具体参照物？（打飞进帐篷/压塌木撑子——环境参与）
- [ ] 声音是否分层出现，顺序是否符合物理逻辑？（骨头→毡布→伤者）
- [ ] 是否有明确的"收着"信号，让读者自己推算出全力是什么？
- [ ] 旁观者反应是否分层？（同伴/士兵/帐篷里的人，各自的反应速度和方式不同）
- [ ] 伤者的具体状态是否写出来了？（血、牙、声音——不用"惨叫"）
- [ ] 胜者打完之后的动作是否延续了他的性格？（步子不停=不把这件事放在眼里）
- [ ] 这场打斗之后，双方的权力关系是否发生了可见的变化？（横挡的人让了半步）
