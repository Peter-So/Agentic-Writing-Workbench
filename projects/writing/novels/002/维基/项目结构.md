# 项目结构 Wiki

本文件记录项目结构性目录和文件的用途。系统执行归档、更新和材料路由时优先读取 `project-structure.json`。

## 结构文件

| 角色 | 文件 | 用途 | 别名/历史名 |
| --- | --- | --- | --- |
| 创作简报 | `简报/创作简报.md` | 片长、核心命题、目标观众、制作限制与整体创作边界。 | `brief/创作简报.md`, `brief.md`, `项目简报.md` |
| 概念 | `开发/概念.md` | 短片概念、logline、主题、主角欲望、选择代价和结尾余味。 | `development/概念.md`, `logline.md`, `概念.md` |
| 角色表 | `人物/角色表.md` | 角色欲望、阻碍、关系、转变和可拍摄行为。 | `characters/角色表.md`, `characters.md`, `角色表.md`, `人物档案.md` |
| 节拍表 | `开发/节拍表.md` | 短片节拍、冲突、转折和画面推进。 | `development/节拍表.md`, `beat_sheet.md`, `节拍表.md` |
| 剧本 | `剧本/剧本.md` | 影视剧本正文，包含场景标题、动作、角色和对白。 | `script/剧本.md`, `screenplay.md`, `script.md`, `剧本.md` |
| 分镜表 | `分镜/分镜表.md` | 镜号、景别、画面、声音、备注和拍摄提示。 | `storyboard/分镜表.md`, `shot_list.md`, `分镜表.md` |
| 影像风格 | `风格/影像风格.md` | 视觉基调、声音策略、剪辑节奏和画面风格。 | `style/影像风格.md`, `style.md`, `影像风格.md` |

## 结构目录

| 角色 | 目录 | 用途 |
| --- | --- | --- |
| brief | `简报` | 项目简报与制作限制。 |
| development | `开发` | 概念、节拍和开发材料。 |
| characters | `人物` | 角色设定和角色视觉资产。 |
| script | `剧本` | 剧本正文和版本。 |
| storyboard | `分镜` | 分镜表和镜头设计。 |
| storyboards | `分镜生成` | 分镜提示词、生图和镜头资产。 |
| style | `风格` | 影像风格、声音和剪辑策略。 |
| assets | `资产` | 角色图、参考图和生成图素材。 |
| outputs | `输出` | 确认稿、快照和生成物。 |
| skills | `技能` | 项目私有技能。 |
| wiki | `维基` | 项目级 Wiki。 |

## 路由说明

- 系统写入结构性材料时，应先按 role 查询本文件，再按 aliases/文件名相似度回退。
- 历史项目迁移后，旧文件名会保留在 aliases 中，由本 Wiki 路由到新规范路径。
- 覆盖式修改属于高风险写入，必须经用户显式确认后执行。
