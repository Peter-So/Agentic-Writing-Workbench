# 角色视觉一致性技能卡

来源：技能库检索参考 `qu-skills/skills@character-design-sheet`、`agentara/skills@video-character-design` 的角色设计方向，结合短片影像一致性需求沉淀。

## 角色锚点

每个主要角色必须保留以下锚点，后续剧本、分镜、生图均不得随意改变：

- 年龄段
- 身高体态
- 脸型与五官关键词
- 发型与发色
- 常穿服装
- 代表性道具
- 标志性动作
- 情绪基线
- 色彩锚点

## 角色表输出

```markdown
## 角色名
- 功能：
- 年龄段：
- 外观锚点：
- 服装锚点：
- 道具锚点：
- 标志性动作：
- 欲望：
- 阻碍：
- 秘密：
- 关系：
- 弧光：
- 禁止变动：
```

## 一致性规则

1. 同一角色在不同镜头、不同抽帧中必须保持外观锚点一致。
2. 可以变化的是情绪、姿态、光线、镜头角度；不能变化的是身份识别锚点。
3. 角色服装如需变化，必须在剧本或分镜中说明原因。
4. 生图提示词必须复用角色锚点和风格锚点。
5. 每个主角/重要角色必须在 `assets/characters/<角色>/` 下生成统一四视图提示词和四视图图片。
6. 后续分镜、节拍、关键帧生图中出现该角色时，必须把对应 `assets/characters/<角色>/Image/four_view.png` 加入参考图。

## 角色四视图

每个角色统一生成一张四视图设定图：

- 正面
- 左侧面
- 背面
- 三分之四侧面

四视图用于锁定脸型、发型、身形比例、服装锚点和气质基线，不作为单个节拍的临时图片。
